# <a name="reseller-performance-dashboard-in-partner-center-insights"></a>Панель мониторинга производительности торгового посредника в центре партнеров Insights

**Соответствующие роли**: глобальный администратор | Агент администратора | Средство просмотра отчетов | Средство просмотра отчетов руководителя

Панель мониторинга производительности торгового посредника в центре партнеров Insights содержит общие сведения о производительности различных непрямых посредников непрямого поставщика CSP. Панель мониторинга предоставляет данные о действующих посредниках, объеме выручки, которые они создают, и о том, какие продукты являются доходами. Косвенные поставщики смогут искать конкретного торгового посредника по имени и искать сведения для торгового посредника на панели мониторинга производительности торгового посредника.

Следующие разделы можно просмотреть на панели мониторинга производительности торгового посредника.

- Сводка
- Географическое распределение торговых посредников
- Добавление/обновление торговых посредников 
- Тенденция дохода торгового посредника 
- Производительность торгового посредника по продуктам
- Активные торговые посредники по расположениям партнеров
- Тренд географического распределения доходов
- Производительность торгового посредника по сегменту клиента
- Состояние подписи продуктов для реселлеров

 > [!NOTE]
 > Этот отчет доступен на панели мониторинга Insights. Чтобы просмотреть этот отчет, необходимо назначить определенную роль в центре партнеров, например Глобальный администратор, администратор учетной записи, средство просмотра отчетов или средство просмотра отчетов руководителя. Дополнительные сведения см. в разделе Глобальный администратор вашей организации. определенные типы данных в этом отчете могут также быть доступны только пользователям с правами администратора для просмотра отчетов.

## <a name="summary"></a>Сводка

В разделе "Сводка" представлено представление моментального снимка ключевых показателей эффективности (KPI), связанных с непрямым поставщиком CSP.

- Активные торговые посредники: число посредников, имеющих по крайней мере одну активную подписку в течение этого месяца.

В микродиаграмме представлена тенденция за месяц по отдельным торговым посредникам, которые были активными в течение выбранного диапазона дат.

- Передаваемые торговые посредники: число торговых посредников, которые продавали по крайней мере одну подписку в течение этого месяца. 

В микродиаграмме представлена тенденция за месяц по месячным торговым посредникам, зарегистрированным для выбранного диапазона дат.

- Новые торговые посредники: число торговых посредников, которые начали работать с косвенным поставщиком в течение этого месяца. 

В микродиаграмме представлена тенденция к общему количеству новых торговых посредников за выбранный диапазон дат за месяц.

- Счет на выручку USD: доход в долларах США, определяемый торговыми посредниками в течение этого месяца. 

В микродиаграмме представлена тенденция дохода за месяц за выбранный диапазон дат.

- Раздел выручка от продаж по продуктам содержит месяц, разбитый на месяц по оплачиваемому доходу в долларах США, с разбивкой по продуктам.

## <a name="geographical-spread-of-resellers"></a>Географическое распределение торговых посредников

Представление "* * торговые посредники по географическому представлению" предоставляет географическое распределение торговых посредников. С помощью этого мини-приложения партнеры могут просматривать общее количество **посредников**, **новых торговых посредников** и **выставляемых доходов (USD)**, разбиваемые по различным географическим регионам.

Вы можете выполнить поиск и выбрать страну в сетке, чтобы увеличить расположение на карте. Нажмите на карте кнопку **Home (домой** ), чтобы вернуться к исходному представлению. Наведите указатель мыши на карту, чтобы просмотреть **выставляемый доход (USD)** по странам. Поле счет выставляемого дохода (USD) в сетке доступно для сортировки.

:::image type="content" source="images/pci/pci-res-perf-resel-by-geo-1.png" alt-text="Торговые посредники по географическому ":::

## <a name="resellers-addchurns"></a>Добавление/обновление торговых посредников

Это представление содержит месяц, разбитый на месяц по количеству **новых торговых посредников**, **обработанных торговых посредников** и **существующих торговых посредников**. 

- Новые торговые посредники: число посредников, которые были недавно зарегистрированы косвенным поставщиком в течение выбранного диапазона дат.
- Обработанные торговые посредники: число посредников, у которых нет транзакции за последние шесть месяцев за исключением текущего месяца.
- Существующие торговые посредники: число посредников, которые были переработаны за предыдущий месяц.

:::image type="content" source="images/pci/pci-res-perf-resel-add-churn-1.png" alt-text="Добавление/обновление торговых посредников":::
