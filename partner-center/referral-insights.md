# <a name="get-referral-insights-in-partner-center-and-find-out-how-your-referrals-are-doing"></a>В Центре партнеров вы можете получить полезные сведения об авторах и их деятельности

**Соответствующие роли**

- Администратор авторов

Страница **Co-sell opportunities** (Возможности совместных продаж) в разделе **Аналитика** в меню "Рефералы" позволяет вам оценивать эффективность рефералов. Регулярно просматривайте эти метрики, чтобы определять тенденции или области, требующие внимания, и двигаться к бизнес-целям.

Чтобы просмотреть данные аналитики по возможностям совместных продаж, в меню Центра партнеров выберите **Рефералы > Аналитика > Возможности совместных продаж**.

> [!Important]
> Будет применен фильтр типа данных **Совместные продажи** для всех данных. Отмените фильтр, если вы хотите проанализировать данные по частным и партнерским сделкам.

## <a name="apply-filters"></a>Применение фильтров

В верхней части страницы **Возможности совместных продаж** можно выбрать период, за который требуется отобразить данные. По умолчанию выбрано значение **3 месяца**, но можно выбрать данные за шесть месяцев или один год. Вы также можете выбрать значение **Другое**, чтобы просмотреть данные по всем рефералам за определенный период времени.

Нажмите кнопку **Фильтры**, чтобы открыть панель, которая позволяет отфильтровать все данные на этой странице по имени клиента, стране, типу сделки, направлению сделки, имени решения и состоянию. Ниже приведены подробные сведения о таких фильтрах.

- **Имя клиента.** По умолчанию используется фильтр **Все**, однако вы можете ограничить объем отображаемых данных одним или несколькими клиентами.
- **Страна.** По умолчанию используется фильтр **Все**, однако вы можете ограничить объем отображаемых данных одной или несколькими странами.
- **Deal type** (Тип сделки). По умолчанию используется фильтр **Совместная продажа**, но вы можете установить фильтр "Все" либо ограничить объем отображаемых данных частными или партнерскими сделками.
- **Направление сделки.** По умолчанию используется фильтр **Все**, однако вы можете ограничить объем отображаемых данных **входящими** (полученными) или **исходящими** (отправленными вами) рефералами.
- **Имя решения.** По умолчанию используется фильтр **Все**, однако вы можете ограничить объем отображаемых данных рефералами, которые содержат одно или несколько выбранных решений.
- **Состояние**. По умолчанию используется фильтр **Все**, но вы можете ограничить объем отображаемых данных данными о рефералах, содержащими один или несколько типов состояния, например "Принято", "Отклонено", "Истек срок действия", "Утеряно" и "Заключено".

Сведения на всех приведенных ниже диаграммах отражают диапазон дат и любые выбранные фильтры с учетом приведенных ниже исключений. Кроме того, некоторые разделы позволяют применять дополнительные фильтры, например фильтр по конкретному решению.

## <a name="referrals-summary"></a>Сводка по рефералам

На этой карточке приводятся общие показатели по использованию вами возможностей совместных продаж.

На диаграмме показано общее число сделок, число заключенных и упущенных сделок, а также общая сумма сделок (в долларах США) за выбранный период времени.

Метрики процентного изменения (показаны красным или зеленым цветом со стрелкой) указывают разницу между **последним полным месяцем в выбранном диапазоне дат** и **первым полным месяцем в данном диапазоне**. Предположим, что текущей датой является 15 июня и вы выбрали фильтр **3 месяца**, чтобы отобразить данные за последние три месяца. В данном случае эти метрики будут показывать разницу между маем (последний полный месяц в выбранном периоде) и мартом (первый полный месяц в выбранном периоде). Выбранный диапазон дат — последние **3 месяца**, поэтому будут сравниваться данные за май и за март.

## <a name="conversion-funnel"></a>Воронка конверсии

В этом разделе отображается визуальный индикатор перехода ваших сделок от одного состояния к другому в ходе их жизненного цикла. Вы можете просмотреть весь жизненный цикл сделок с учетом их общей суммы и суммы в долларах США по основной сводке для этого раздела. В первом разделе приведены данные о состоянии и о типе сделки, благодаря чему вы можете визуально оценить объем или сумму по типу. Также доступен раздел **Рефералы из прошлого**, который используется для обозначения сделок, по которым вы предприняли действия (принятие или отклонение) или которые вы пометили как заключенные или утраченные за выбранный для отчета период времени. Вы можете применить фильтры для просмотра динамики по сделкам на различных этапах их жизненного цикла.

Входящие сделки совместной продажи могут быть объединены в принятые, отклоненные или просроченные, так как партнеры должны принять или отклонить такие входящие сделки.

:::image type="content" source="images/referrals/inbound.png" alt-text="Изображение: состояние входящих рефералов.":::

Исходящие частные, партнерские сделки и сделки совместной продажи будут объединены в созданные, так как сделки такого типа создаются партнерами.

## <a name="deals-by-solutions"></a>Сделки по решениям

На этой диаграмме можно увидеть, какие решения генерируют наибольшее количество рефералов и наибольшую сумму сделок. Таблица включает три сводки: совместные продажи, партнерские сделки и частные сделки.
В зависимости от выбора сведения вы можете просмотреть показатели по сделкам, агрегированным по решению.

> [!NOTE]
> Если в сделку включено несколько решений, в таблице будет отображена одна сделка с учетом для всех таких решений. Не суммируйте значения, связанные с решениями, и не сравнивайте их с другими метриками объема рефералов. Это представление призвано помочь вам определить эффективность сделки со сведением по решениям.

Таблица включает сведения об общем количестве сделок с решением, а также о соответствующих состояниях, таких как заключенные сделки, утраченные сделки, сделки с истекшим сроком, общая сумма заключенных и утраченных сделок в долларах США. Справа от таблицы также приведена диаграмма динамики с отображением общего числа сделок и общей суммы заключенных сделок в долларах США на основе выбранного решения. По умолчанию выбрано значение "Все решения".

Три показателя, по которым вы можете сравнить данные

- Тип сделки
- Рынки
- Решения

При выборе типа сделки вы можете сравнить эффективность рефералов в отношении возможностей совместных продаж, частных и партнерских сделок. Справа от каждой линейчатой диаграммы также отображается круговая диаграмма с распределением этих же данных по процентной доле.

## <a name="no-data"></a>Нет данных

Существует несколько причин, по которым диаграмма может быть пустой (как показано ниже) при использовании аналитики совместных продаж.

- Отсутствуют данные по этому клиенту. Попробуйте создать сделки, чтобы заполнить этот отчет.
- Возникла проблема с подключением к сети. Проверьте подключение к Интернету и повторите попытку.
- Страница загружается с фильтром по умолчанию для сделок совместных продаж. Если у вас есть только частные сделки, сбросьте фильтр типа сделок.
- Отсутствуют записи, соответствующие примененному фильтру. Попробуйте сбросить фильтры.
- Существует задержка между изменением состояния возможности и его обновлением в отчете аналитики. Проверьте отчет через 24 часа.

:::image type="content" source="images/referrals/nodata.png" alt-text="Изображение с отсутствием визуализации данных для рефералов.":::

> [!NOTE]
> На странице **Сведения о реферале** отображаются только данные авторов, созданных в Центре партнеров. На ней не отображаются данные авторов, созданных с помощью [Partner Sales Connect](psc-to-pc.md) или других механизмов.
