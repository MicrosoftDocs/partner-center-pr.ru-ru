# <a name="view-manage-and-export-your-incentives-program-details"></a>Просмотр, управление и экспорт сведений о программе поощрения

**Относится к**

- Центр партнеров

**Соответствующие роли**

- Администратор поощрений
- Пользователь программы поощрений
- Поощрение средства просмотра только для чтения
- Глобальный администратор
- администратор партнера MPN.

Используйте эти страницы для просмотра сведений о прошлых, ожидающих и будущих платежах и доходах, о состоянии программ поощрения в каждом местоположении, а также о полном журнале транзакций. Вы также можете скачать или экспортировать подробные сведения для любой транзакции. На всех страницах поощрения отображаются суммы в долларах США.

>[!NOTE]
>Вы увидите только данные для идентификаторов и программ MPN, с которыми вы связаны. Если вы хотите получить доступ к дополнительным данным, обратитесь к администратору поощрения. 

## <a name="access-the-incentives-pages"></a>Доступ к страницам поощрения

Получить доступ к информации о поощрениях можно несколькими способами с [панели мониторинга](https://partner.microsoft.com/dashboard)центра партнеров:

- Выберите **поощрения** в меню слева. В разделе **поощрения**, включая **Обзор**, отображается несколько параметров. Изучите эти возможности, чтобы узнать о состоянии поощрения программ, поощрительных платежах, прибыли и заявках.
- Щелкните значок **money** (выплата) в правом верхнем углу. Затем можно выбрать другие параметры, такие как **журнал транзакций**, **платежи**, **Экспорт данных**или **отчет о продажах Office**.

Здесь описаны следующие страницы:

- Моя страница обзора поощрения
- Страница "Платежи"
- Страница "Журнал транзакций"
- Страница экспорта данных и другие связанные страницы экспорта
- Сводный отчет по транзакциям

>[!NOTE]
>Дополнительные сведения о поощрениях и стимулах в центре партнеров см. в разделе [инвестиции и поощрения партнеров](https://partner.microsoft.com/membership/partner-incentives) (требуется вход в систему).

## <a name="my-incentives-overview-page"></a>Моя страница обзора поощрения

Страница обзора **My incentives** (Мои поощрения) отображает общее состояние программ поощрений, а также каждой отдельной программы в каждом расположении. Сюда включены такие сведения, как общие доходы и выплаты по программе поощрений с момента регистрации в программах поощрений.

На этой странице также отображаются сведения о доходах и платежах по расположению или программе, а также дополнительные действия, которые могут потребоваться для регистрации в программе в определенном расположении.

Чтобы получить доступ к странице обзора **My incentives** (Мои поощрения), сделайте следующее.

1. Войдите на [панель мониторинга](https://partner.microsoft.com/dashboard) Центра партнеров.
2. Выберите **поощрения**, а затем — **Обзор** в меню.
3. Просмотрите сводные сведения о доходах и платежах в верхней части страницы и (или) более подробные сведения в таблице ниже. Кроме того, можно сортировать, группировать и расширять сопутствующую таблицу:

   - Для сортировки по столбцу выберите имя столбца.
   - Для группировки по программе выберите вкладку **по программе** над таблицей.
   - Чтобы выполнить группировку по расположению, выберите вкладку **по расположению** над таблицей.
   - Чтобы просмотреть дополнительные сведения о регистрации в определенной группе, щелкните символ шеврона в конце данной строки. Этот Шеврон расширяет представление.

4. Если для регистрации в программе требуется дополнительное действие, эта информация появится в столбце **Status** (Состояние). В этом случае щелкните символ шеврона, чтобы узнать о дальнейших действиях.

>[!NOTE]
>Дополнительные сведения о различных состояниях регистрации, отображаемых в столбце **состояние** , см. в следующей таблице.

Дальнейшие действия или ссылки для обновления банковских или налоговых сведений в центре партнеров см. в столбце **необходимые действия** . |

## <a name="transaction-history-page"></a>Страница "Журнал транзакций"
На этой странице приводятся все доходы, включая дату, тип и размер. Вы можете выбрать период времени для просмотра, а также выполнить фильтрацию по идентификатору регистрации, программе, идентификатору выплаты, типу прибыли, рычагу и состоянию. Данные доступны для текущего финансового года (1 июля – 30 июня) и предыдущих двух финансовых годов.

Чтобы просмотреть дополнительные сведения о прибыли, щелкните стрелку вниз в правой части страницы. Это приведет к отображению рычага, суммы дохода, продукта и клиента. Если по какой-либо причине эти данные недоступны, но вам требуется доступ к ней, обратитесь в службу поддержки. Если получение является результатом корректировки, а не транзакции, поля продукт и клиент не будут отображаться.

Чтобы экспортировать какие-либо данные транзакции на этой странице, выберите **Экспорт** , а затем следуйте указаниям на странице Экспорт данных. В файлах, экспортированных со страницы "Журнал транзакций", данные приводятся в валюте транзакции, доходы — в валюте транзакции и долларах США, а сумма выплат — в валюте оплаты.

## <a name="export-data-page"></a>Страница "Экспорт данных"
Следуйте инструкциям на этой странице, чтобы экспортировать нужные данные.

**Примечания.**

- Учитывая большой объем данных, соглашение об уровне обслуживания для отправленного запроса будет обработано 24 часа.
- Вы увидите только данные для идентификаторов и программ MPN, с которыми вы связаны. Если вы хотите получить доступ к дополнительным данным, обратитесь к администратору поощрения.
- Страница "Экспорт данных" не обновляется самостоятельно. Для отображения последних данных может потребоваться обновить страницу вручную.
- Фильтр может вызвать ошибку **Данные недоступны**. Возможно, эта ошибка означает, что вы оставили период по умолчанию, выбранный в течение трех месяцев, а затем выделили идентификатор оплаты из него за пределами этого периода. Расширьте период времени и повторите попытку.

## <a name="transaction-summary-report"></a>Сводный отчет по транзакциям

>[!NOTE]
>Сводный отчет по транзакциям не будет показывать все ваши программы или доходы. Это сводка, сосредоточенная только на обобщенных рычагах. Чтобы просмотреть отдельные транзакции, скачайте [отчет журнала транзакций](#transaction-history-page).

Сводный отчет по транзакциям содержит сведения о транзакциях сводного уровня, включая имена клиентов и идентификаторы подписок, для агрегированных рычагов. Отчет включает каждый идентификатор, связанный с несколькими транзакциями. Общая сумма получения и прибыль для этого идентификатора получения повторяются для каждого клиента или подписки. Это обусловлено тем, что стимулы для обобщенных рычагов послужили совокупно, а не за подписку.


### <a name="download-the-transaction-summary-report"></a>Скачать сводный отчет по транзакциям

1. На странице **Обзор поощрения** выберите значок **money** (выплата) в правом верхнем углу.
2. Выберите пункт **Экспорт данных**.
3. Перейдите на вкладку **Сводка транзакций** .
4. Выберите соответствующие фильтры, чтобы просмотреть программы и рычаги, доступные для агрегированного дохода. Если данные не отображаются, ваши программы не подходят для агрегированных рычагов.
5. Нажмите кнопку **начать скачивание**   , чтобы экспортировать отчет.  

>[!NOTE]
>Сводный отчет по транзакциям не будет включать сведения о клиентах и подписках для подписок более чем за 60 дней, или если доход потребления меньше одного цента.
