## <a name="updated-promotions-for-office-365"></a><a id="14"/></a>Обновлены рекламные акции для Office 365

### <a name="categories"></a>Категории

- Дата: 30.04.2020
- Введите команду: Характеристики
- Состояние Ожидается в ближайшее время
- [Подробнее](https://aka.ms/CSPCOVIDPromo)

### <a name="summary"></a>Сводка

4 мая 2020 г. в 12:01 (UTC) корпорация Майкрософт объявила о новых акциях, связанных с Office 365 E1 и Microsoft 365 бизнес базовый, чтобы заменить текущую 6-месячную пробную версию Office 365 E1 для новых клиентов.

### <a name="impacted-audience"></a>Затронутая аудитория

Все партнеры, заключающие сделки в рамках программы "Поставщик облачных решений" (CSP). Это специальное предложение доступно только для новых коммерческих клиентов.

### <a name="details"></a>Подробности

В рамках [обязательства корпорации Майкрософт перед клиентами](https://www.microsoft.com/microsoft-365/blog/2020/03/05/our-commitment-to-customers-during-covid-19/) в это беспрецедентное время шестимесячная пробная версия Office 365 E1 будет доступна новым клиентам в рамках CSP (см. определение ниже).
4 мая 2020 г. корпорация Майкрософт предложила новые скидки на Office 365 E1 и Microsoft 365 бизнес базовый, чтобы новые клиенты могли бесплатно использовать эти продукты в течение первых шести месяцев в рамках своей годовой подписки с ежемесячной оплатой.

Эти акции заменят текущее предложение пробной версии Office 365 E1. Отличаются они следующим образом:

- Партнеры будут заключать ежегодные сделки не на использование уникальной шестимесячной пробной версии SKU, а на использование SKU Office 365 E1 или Microsoft 365 бизнес базовый по акции в рамках CSP, когда 100 % скидка предоставляется в течение первых шести месяцев. [Подробнее об акциях CSP.](https://docs.microsoft.com/partner-center/promotions)
- Возможность заказов будет технически ограничена только новыми клиентами, а не пробной версией Office 365 E1, которая была ограничена только новыми клиентами в рамках описанной политики. См. [вопросы и ответы о партнерах Майкрософт](https://aka.ms/CSPCOVIDPromo).
- Для клиентов SMB предложение Microsoft 365 бизнес базовый будет включено в эту акцию. Это гарантирует, что партнеры смогут перевести клиентов на Microsoft 365 для бизнеса при наличии менее чем 300  мест в нужном семействе продуктов.
Клиенты с текущей лицензией на пробную версию Office 365 E1 (доступна до 4 мая 2020 г.) будут использовать свой план и дальше. Это изменение влияет только на новых клиентов.

Определение новых клиентов

Новые клиенты, на которых распространяется эта акция:

1. Новые арендаторы клиента.
2. Существующие клиенты, в организации-арендаторе которых сейчас нет платной или недавно отмененной (в течение последних 30 дней) подписки на любой из продуктов, определенных в [разделе с вопросами и ответами о партнерах Майкрософт](https://aka.ms/CSPCOVIDPromo).

### <a name="additional-resources"></a>Дополнительные ресурсы

- Дополнительные сведения об обеспечении безопасной удаленной работы см. в [центре ресурсов удаленной работы](https://www.microsoft.com/microsoft-365/partners/remotework).
- Сведения об этих изменениях см. в [разделе вопросов и ответов для партнеров](https://aka.ms/CSPCOVIDPromo).

### <a name="next-steps"></a>Дальнейшие действия

Ознакомьтесь с описанными изменениями и прейскурантом за май 2020 года и Передайте эти сведения всем соответствующим лицам в своей организации.

## <a name="azure-reservations-new-customer-self-serve-capability-with-partner-permission"></a><a id="13"/></a>Возможность самостоятельного резервирования продуктов Azure новыми клиентами с разрешениями партнеров

### <a name="categories"></a>Категории

- Дата: 29.04.2020
- Введите команду: Характеристики
- Состояние Уже доступно
- [Подробнее](https://partner.microsoft.com/resources/collection/customer-permission-to-purchase#/)

### <a name="impacted-audience"></a>Затронутая аудитория

Партнеры по программе "Поставщик облачных решений" (CSP)

### <a name="summary"></a>Сводка

Корпорация Майкрософт стремится обеспечить гибкость и предоставить различные варианты, чтобы упростить приобретение и использование продуктов и услуг от Майкрософт и наших партнеров.

В рамках нашего постоянного обязательства мы представляем новую возможность самостоятельного резервирования продуктов Azure для клиентов, которая упрощает процесс приобретения и дополнительно улучшает использование резервирований Azure через партнеров по программе CSP.

Резервирования Azure — это отличный способ приобрести службы Azure со скидкой. Благодаря этой новой возможности партнеры могут предоставить клиентам разрешение на гибкое приобретение используемых ими ресурсов в рамках существующих подписок Azure, которые они уже приобрели для своих клиентов.

Вот некоторые сведения об этой возможности:

- Партнеры могут включить эту возможность на уровне учетной записи отдельного клиента и предоставить клиенту контроль над покупкой.
- По умолчанию эта возможность отключена, а партнер может включить ее и предоставить клиенту разрешение.
- Партнеры смогут проверять покупки клиента и управлять ими в Центре партнеров.
- С помощью интерфейсов API можно включить разрешение для клиентов в большом масштабе и автоматизировать процесс.

### <a name="next-steps"></a>Дальнейшие действия

1. Помогите клиентам зарезервировать используемые ими службы Azure с помощью новой возможности.
2. [Ознакомьтесь с ресурсами из этой коллекции](https://partner.microsoft.com/resources/collection/customer-permission-to-purchase#/), которые содержат дополнительные сведения, в том числе с [технической документацией](https://docs.microsoft.com/partner-center/give-customers-permission), а также с пошаговыми руководствами для партнеров и клиентов.

## <a name="microsoft-reversed-the-csp-eur-price-for-microsoft-365-e3"></a><a id="12"/></a>Корпорация Майкрософт возвращает цену в евро на Microsoft 365 E3 в рамках CSP.

### <a name="categories"></a>Категории

- Дата: 24.04.2020
- Введите команду: Предложения и рынки
- Состояние Уже доступно
- [Подробнее](https://flow.microsoft.com/ui-flows/)

### <a name="summary"></a>Сводка

1 апреля 2020 г. корпорация Майкрософт исправила ошибку в ценах в евро на Microsoft 365 (M365) E3. Однако 22 апреля 2020 г. корпорация Майкрософт временно вернула предыдущие цены, чтобы помочь партнерам в этих чрезвычайных обстоятельствах.

### <a name="impacted-audience"></a>Затронутая аудитория

Все партнеры, предлагающие M365 E3 согласно прейскуранту в евро в рамках программы "Поставщик облачных решений".

### <a name="details"></a>Подробности

Ранее в этом году мы обнаружили ошибку в ценах в евро на Microsoft 365 E3 в рамках CSP: цена нетто была меньше, чем следует. Мы запланировали обновление системы, чтобы исправить цену, изменив стоимость предложения с 1 апреля 2020 г.

Хотя прежняя цена была указана по ошибке, исправлять ее и повышать цены сейчас неуместно, учитывая чрезвычайные обстоятельства. Поэтому с 22 апреля 2020 г. мы временно возвращаем предыдущую, более низкую цену нетто. Предыдущая, более низкая цена будет немедленно отображена во всех счетах-фактурах, а в прейскурантах она появится в мае 2020 года, в ближайшем будущем.

Партнеры не должны использовать цены на M365 E3 в евро, указанные в прейскуранте за апрель 2020 года. Вместо него следует использовать прейскурант за март 2020 года.

Мы признательны сообществу наших партнеров за их отличную работу в это беспрецедентное время.

### <a name="next-steps"></a>Дальнейшие действия

Передайте эти сведения всем соответствующим лицам в своей организации.

### <a name="questions"></a>Есть вопросы?

Если вам нужна помощь по работе с Майкрософт, обращайтесь в службу поддержки, сотрудники которой ответят на ваши вопросы и запросы, а также предоставят подробную информацию.

## <a name="reminder-activation-of-security-safeguards-for-partner-center-transactions-in-csp"></a><a id="11"/></a>НАПОМИНАНИЕ. Активация мер безопасности для транзакций Центра партнеров в рамках программы CSP

### <a name="categories"></a>Категории

- Дата: 23.04.2020
- Введите команду: Характеристики
- Состояние Ожидается в ближайшее время
- [Подробнее](https://partner.microsoft.com/resources/collection/partner-security-requirements#/)

### <a name="target-partner-audiences"></a>Целевая аудитория из числа партнеров

- Все партнеры, участвующие в программе "Поставщик облачных решений" (CSP).
- Все поставщики панелей управления.
- Все партнеры, участвующие в программе для консультантов

### <a name="summary"></a>Сводка

Повышение и поддержание уровня безопасности и конфиденциальности входит в число наших основных приоритетов, поэтому мы продолжаем оказывать партнерам помощь в защите клиентов и арендаторов. В 2019 году мы ввели [обязательные требования к безопасности](https://blogs.partner.microsoft.com/mpn/safeguard-business-security-best-practices/). Всем партнерам, участвующим в программе "Поставщик облачных решений" и программе для консультантов, а также поставщикам панелей управления следует выполнить эти требования, чтобы обеспечить дальнейшее нормативное соответствие.

Недавно мы успешно завершили активацию мер безопасности для возможностей администрирования с делегированными правами (AOBO) для всех арендаторов партнеров, обеспечив защиту их организаций и клиентов от инцидентов, связанных с кражей личных сведений.

Для дополнительной защиты партнеров и клиентов **1 мая 2020 г. мы начнем активацию мер безопасности для транзакций Центра партнеров в рамках CSP**. Активация дополнительных мер безопасности означает, что от партнеров будет требоваться применять многофакторную проверку подлинности (MFA) для любых транзакций в рамках CSP с использованием интерфейса или API Центра партнеров. Это позволит предотвратить несанкционированный доступ и обеспечит защиту клиентов и арендаторов от инцидентов безопасности, связанных с кражей личных сведений. Дополнительные сведения см. [здесь](https://docs.microsoft.com/partner-center/partner-security-requirements-mandating-mfa).

### <a name="next-steps"></a>Дальнейшие действия

- **Если вы не активировали рекомендуемые требования к безопасности для защиты арендаторов, с которыми работаете, и их пользователей**, сделайте это немедленно. Это позволит обеспечить дальнейшее нормативное соответствие и избежать перебоев в работе, связанных с управлением клиентами или транзакциями в рамках CSP. Партнеры, которые не выполнят требования к безопасности, подвергнут риску свое участие в программе CSP.
- Непрямым поставщикам следует выполнить эти требования к безопасности в своих организациях, а также провести работу с торговыми посредниками, чтобы обеспечить выполнение этих требований в соответствующих организациях, с помощью этого шаблона письма. Мы также напрямую свяжемся с вашими торговыми посредниками по электронной почте.
- В [Центре партнеров ознакомьтесь со сведениями на странице состояния соответствия требованиям к безопасности](https://partner.microsoft.com/pcv/security/compliance), чтобы узнать о своих результатах и важных действиях, которые следует предпринять. Регулярно обращайтесь к глобальному администратору, чтобы проверять выполнение требований к безопасности для каждого из своих арендаторов.
- Если вы используете решение поставщика панели управления (CPV), обратитесь к нему за консультацией.

## <a name="microsoft-customer-agreement-standard-amendments-have-moved"></a><a id="10"/></a>Стандартные изменения к Клиентскому соглашению Майкрософт перемещены

### <a name="categories"></a>Категории

- Дата: 21.04.2020
- Введите команду: Характеристики
- Состояние Уже доступно

### <a name="impacted-audience"></a>Затронутая аудитория
- Все партнеры CSP.

### <a name="summary"></a>Сводка

Стандартные изменения к Клиентскому соглашению Майкрософт, которые ранее были доступны на странице с ценами и предложениями в Центре партнеров, перемещены. Теперь партнеры могут скачать изменения в разделе поддержки Клиентского соглашения Майкрософт в Центре партнеров (только выполнив безопасный вход). Предварительное утверждение от корпорации Майкрософт для клиентов не требуется. Это касается изменения к соглашению о финансовых услугах и других стандартных изменений к Клиентскому соглашению Майкрософт.

Перемещение не повлияло на содержимое изменений.

Стандартные изменения к Клиентскому соглашению Майкрософт:

- по всему миру (английский) — изменение к соглашению о финансовых услугах;
- Аргентина (английский) — изменение к соглашению об обработке данных для клиентов из Аргентины;
- Аргентина (испанский) — изменение к соглашению об обработке данных для клиентов из Аргентины;
- Германия (английский) — изменение к соглашению о профессиональной тайне для клиентов из Германии;
- Германия (немецкий) — изменение к соглашению о профессиональной тайне для клиентов из Германии;
- США (английский) — изменение к соглашению с деловым партнером о HIPAA для Microsoft Online Services.

Как партнеру получить доступ к изменениям?
Чтобы получить доступ к изменениям, выполните следующие действия:

- Шаг 1. На панели навигации слева выберите элемент "Поддержка" и создайте запрос на поддержку.
- Шаг 2. Введите текст "Клиентское соглашение Майкрософт", чтобы найти нужный раздел.
- Шаг 3. Выберите его и щелкните элемент "Обзор решений".
- Шаг 4. Щелкните требуемое изменение.

Или же войдите в Центр партнеров и перейдите по [этому URL-адресу](https://partner.microsoft.com/dashboard/support/csp/servicerequests/create?stage=2&topicid=c007d538-f817-84e3-06b5-ff4005c1a6bb).

На других рынках налоги применяются в соответствии с обычной процедурой выставления счетов. Прежде чем принимать решение о повышении цены на SKU "Голосовая связь Microsoft 365 бизнес", обратитесь к юрисконсульту по поводу освобождения от уплаты налогов. Если цена на SKU "Голосовая связь Microsoft 365 бизнес" повышена, партнеру необходимо найти способ освобождения от уплаты налогов на телекоммуникационные услуги. В противном случае партнер не будет оплачивать добавочный налог на повышение цены, что приведет к нарушению налогового законодательства. Корпорация Майкрософт предоставляет дополнительные поощрения партнерам в рамках CSP, которые предлагают SKU "Голосовая связь Microsoft 365 бизнес" в США, чтобы компенсировать потенциальное отсутствие прибыли.

### <a name="guidance-for-partners-in-australia"></a>Рекомендации для партнеров из Австралии

Партнеры из Австралии должны учитывать, что предложение "Голосовая связь Microsoft 365 бизнес без тарифного плана" не будет доступно в рабочей версии прейскуранта 1 апреля 2020 г. несмотря на то, что оно доступно в предварительной версии прейскуранта в марте 2020 г. Благодарим вас за понимание.

Все партнеры CSP.

**Обновление от 1 апреля 2020 г.** По состоянию на 31 января 2020 г. все партнеры в рамках программы "Поставщик облачных решений" (CSP) должны были принять условия соглашение для всех своих клиентов, ведущих деятельность в рамках CSP. Ранее мы сообщали, что следующий этап реализации соглашения начнется в апреле 2020 года.  Чтобы исключить перебои в работе, мы решили продлить следующий этап реализации, не ограничивая его апрелем 2020 года. Некоторые партнеры в рамках программы "Поставщик облачных решений" сообщают, что у них возникли проблемы с принятием условий MPA. Мы работаем над обеспечением ресурсов Майкрософт для партнеров, у которых возникли проблемы с принятием условий соглашения.

### <a name="summary"></a>Сводка
С 31 января 2020 г. всем партнерам в рамках программы "Поставщик облачных решений", в том числе непрямым торговым посредникам, следует принять условия Соглашения с партнером Майкрософт, чтобы не потерять возможность создавать новых клиентов или размещать новые заказы для существующих клиентов в рамках программы CSP. Теперь принятие условий Соглашения с партнером Майкрософт будет обязательным требованием для всех партнеров в рамках программы CSP.

С апреля 2020 г.:
- Партнеры с прямым выставлением счетов и непрямые поставщики, которые не приняли условия Соглашения с партнером Майкрософт, не смогут размещать новые заказы и смогут управлять только существующими подписками.
- Непрямые торговые посредники, которые не приняли условия Соглашения с партнером Майкрософт, потеряют право действовать в качестве зарегистрированных партнеров в новых заказах.

### <a name="actions-for-indirect-providers"></a>Действия для непрямых поставщиков

2. Примите условия Соглашения с партнером Майкрософт на [странице с общими сведениями о CSP в Центре партнеров](https://partner.microsoft.com/pcv/dashboard/overview) с помощью учетных данных глобального администратора.
3. [Проверьте состояние подписания соглашения своими непрямыми торговыми посредниками с помощью API.](https://docs.microsoft.com/partner-center/develop/verify-indirect-reseller-mpa-status)
4. Помогите непрямым торговым посредникам выполнить действия, указанные для них ниже.
