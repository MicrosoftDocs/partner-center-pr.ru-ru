## <a name="deprecation-and-retirement-of-existing-get-and-put-qualification-apis-for-the-education-customer-validation-process-by-the-end-of-february-2021"></a><a name="2"></a>Вывод из эксплуатации и прекращение поддержки существующих API GET и PUT для проверки соответствия требованиям для процесса проверки клиентов в конце февраля 2021 г.

### <a name="impacted-audience"></a>Затронутая аудитория

Партнеры, продающие предложения для образовательных учреждений в рамках программы "Поставщик облачных решений" (CSP), и использующие API Центра партнеров.

### <a name="details"></a>Сведения 

Это дополнение к улучшениям API, выпущенным в декабре 2020 г. В декабре 2020  г. были выпущены новые API-интерфейсы GET и POST для проверки соответствия требованиям. Старые API-интерфейсы GET и PUT для проверки соответствия требованиям будут выведены из эксплуатации в конце февраля 2021 г. После этого вам нужно будет использовать новые API-интерфейсы Центра партнеров GET и POST, чтобы иметь возможность приобретать предложения для образовательных учреждений. 

## <a name="next-steps"></a>Дальнейшие действия

- Выполните обновление до новых API-интерфейсов (если еще не сделали этого) для успешного и своевременного перехода.
- Ознакомьтесь со сведениями об изменениях API-интерфейсов в Центре партнеров и [руководстве в ресурсах по подготовке к работе:  "Улучшения процесса проверки клиентов в Центре партнеров"](https://partner.microsoft.com/resources/collection/partner-center-edu-validation-enhancements#/).
- Передайте эти сведения соответствующим командам в организации и торговым посредникам, чтобы помочь им подготовиться к изменениям.

## <a name="dynamics-365-offers-for-february-2021"></a><a name="1"></a>Предложения Dynamics 365 на февраль 2021 г.

### <a name="categories"></a>Категории

- Дата: 04.01.2021
- Предложения

### <a name="summary"></a>Сводка

В феврале 2021 г. будут запущены новые предложения Dynamics 365.

### <a name="impacted-audience"></a>Затронутая аудитория

Партнеры, участвующие в программе "Поставщик облачных решений" (CSP), программах для партнеров с прямым выставлением счетов, косвенных поставщиков и программах поощрений косвенных торговых посредников

### <a name="details"></a>Сведения

Корпорация Майкрософт объявляет о предстоящих изменениях в предложениях Dynamics 365, которые будут запущены в феврале 2021 г.:

- предложение дополнительных ответов для Dynamics 365 Customer Voice;
- прекращение поддержки Dynamics 365 Customer Service Insights;
- изменение идентификаторов предложения Dynamics 365 Cloud AX Migration.

**Предложение дополнительных ответов для Dynamics 365 Customer Voice**

1 февраля 2021 г. корпорация Майкрософт создаст новое предложение дополнительных ответов, предоставляющее право на получение 1000 ответов, которое заменит текущее предложение с правом на получения 2000 ответов на опрос, которое будет действовать до 1 февраля 2021 г.

Сведения о номере SKU предложения см. на вкладке Customer Voice Addl Resp в [документе Excel по предложениям Dynamics в рамках CSP за февраль 2021 г.](https://partner.microsoft.com/resources/detail/dynamics-csp-offers-february-2021-xls) Дополнительные сведения см. на [домашней странице Microsoft Dynamics 365 Customer Voice](https://dynamics.microsoft.com/en-us/customer-voice/overview/).

**Прекращение поддержки Dynamics 365 Customer Service Insights**

1 февраля 2021 г. будет прекращена поддержка следующих продуктов:

- Dynamics 365 Customer Service Insights (включая возможность добавления дополнительных случаев);
- Dynamics 365 Virtual Agent for Customer Service

Возможности Customer Service Insights будут перенесены в Dynamics 365 Customer Service, где клиенты смогут найти те же функции для обслуживания клиентов.  

Сведения о номере SKU предложения см. на вкладке Customer Service Insights в [документе Excel по предложениям Dynamics в рамках CSP за февраль 2021 г.](https://partner.microsoft.com/resources/detail/dynamics-csp-offers-february-2021-xls) Дополнительные сведения см. на [домашней странице Microsoft Dynamics Customer Service](https://dynamics.microsoft.com/customer-service/overview/).
