## <a name="partner-center-software-license-key-and-download-fulfillment-notice"></a><a name="17"></a>Уведомление об обработке лицензионных ключей и скачивании программного обеспечения в Центре партнеров

### <a name="categories"></a>Категории

- Дата: 30.09.2020
- Характеристики
 
### <a name="impacted-audience"></a>Затронутая аудитория

Все партнеры — поставщики облачных решений (CSP) с предложениями программного обеспечения.

### <a name="details"></a>Подробнее

Сейчас партнеры могут использовать Центр партнеров или Центр администрирования Microsoft 365 для скачивания программного обеспечения и обработки лицензионных ключей для предложений программного обеспечения от партнеров CSP. С **19 января 2021 г.** возможности Центра партнеров по обработке лицензионных ключей и скачиванию программного обеспечения станут недоступны. Такие задачи партнеры будут выполнять исключительно в **Центре администрирования Microsoft 365**. На странице "Пользовательское программное обеспечение" в Центре партнеров вы найдете прямую ссылку на Центр администрирования Microsoft 365, где сможете получить пользовательские лицензии и ссылки на скачивание программного обеспечения. Это позволит обеспечить максимальную безопасность для клиентов и эффективность их работы.

Лицензионные ключи и ссылки на скачивание программного обеспечения — это ценные и востребованные ресурсы интеллектуальной собственности. В случае их утечки доступные активации могут быстро закончиться, а работа клиентов и партнеров ухудшится. Поэтому очень важно, чтобы партнеры использовали самые безопасные технологии и лучшие методики для защиты таких ценных ресурсов и своих клиентов.

**Центр администрирования Microsoft 365**

Центр администрирования Microsoft 365 расширяет возможности по безопасной доставке приобретенного программного обеспечения и позволяет клиентам управлять своими подписками и самостоятельно выделять их при необходимости, что упрощает их работу. Он также сокращает риски, связанные с копированием и отправкой лицензионных ключей непосредственно клиенту, как и при использовании Центра партнеров. В качестве дополнительной меры безопасности все действия партнеров по доставке ключей и файлов программного обеспечения для скачивания и по доступу к ним отслеживаются в журнале действий в Центре партнеров. Благодаря такой функции партнеры могут отслеживать доступ внутренних пользователей для создания заказов, получения ключей и файлов для скачивания, а также расширить возможности контроля и сократить потенциальные риски мошенничества.

### <a name="next-steps"></a>Дальнейшие действия

- Дополнительные сведения о программном обеспечении Центра администрирования Microsoft 365 см. в статьях с [обзором](https://support.microsoft.com/office/admin-center-overview-fa715fa8-6eda-456d-b177-d1c156edd1b3), информацией об [общих задачах](/microsoft-365/admin/admin-overview/about-the-admin-center?view=o365-worldwide&preserve-view=true) и [возможностях обработки лицензионных ключей](https://partner.microsoft.com/resources/detail/microsoft-365-admin-center-software-key-download-pdf).

## <a name="availability-of-new-subscriptions-transition-tool-between-partners-for-azure-plan"></a><a name="15"></a>Доступность нового средства для переноса подписок между партнерами в рамках плана Azure 

### <a name="categories"></a>Категории

- Дата: 28.09.2020
- Характеристики

### <a name="target-audience"></a>Целевая аудитория

- Партнеры CSP с прямым выставлением счетов и косвенные поставщики получат доступ к средству для переноса.  
- Торговые посредники могут обратиться к косвенному поставщику. Дополнительные сведения для поставщиков доступны в ресурсах [здесь](https://partner.microsoft.com/resources/collection/transition-tool-azure-subscriptions-new-commerce-experience-csp#/). 

### <a name="summary"></a>Сводка

С 30 сентября 2020 г. новое средство для переноса, позволяющее автоматизировать перенос подписок Azure клиента между партнерами CSP, станет доступно в рамках новой коммерческой модели для Azure (плана Azure).

Расширяя возможности новой коммерческой модели для Azure, средство перехода повышает гибкость при управлении подписками Azure и предоставляет следующие преимущества для поддержки меняющихся бизнес-потребностей клиентов:  
- экономия времени благодаря удобному средству, автоматизирующему переход в Azure;  
- отсутствие прерываний в работе служб Azure во время переноса права на выставление счетов. 

### <a name="key-capabilities-and-considerations"></a>Основные возможности и рекомендации

- Для перехода нужно решение клиента, и он выполняется только тогда, когда оба партнера предпринимают явные действия.  
-   Переход выполняется мгновенно без какого-либо простоя для клиентов и воздействия на службы Azure. 
-   Возможности этого средства в Центре партнеров станут доступны для партнеров с прямым выставлением счетов и косвенных поставщиков. Торговым посредникам необходимо будет обратиться к косвенным поставщикам, которые будут использовать это средство от их имени. 

### <a name="next-steps"></a>Дальнейшие действия 

-   Изучите ресурсы на [этой странице коллекции для партнеров](https://partner.microsoft.com/resources/collection/transition-tool-azure-subscriptions-new-commerce-experience-csp#/). 
-   Обучите своих специалистов работе с новым средством перехода и приступайте к работе с ним. 
-   Если вы являетесь косвенным поставщиком, обеспечьте контроль над процессом перехода для своих торговых посредников. 
-   Если у вас возникли дополнительные вопросы, обратитесь в службу поддержки.

Партнеры, продающие предложения для образования в рамках программы "Поставщик облачных решений" (CSP).

## <a name="details"></a>Подробнее

Корпорация Майкрософт работает на основе принципов доверия, и мы стремимся обеспечить соответствующий требованиям, безопасный и защищенный метод проверки клиентов для работы с предложениями для образовательных учреждений по программе "Поставщик облачных решений" (CSP). Они повысят наглядность и расширят возможности визуального контроля при проверке клиентов, а также позволят вводить более точные данные, что повысит вероятность успешного прохождения проверки клиентами.

- Улучшения пользовательского интерфейса также повысят точность записи данных и улучшат процесс проверки клиентов из сферы образования корпорацией Майкрософт.

Мы настоятельно рекомендуем использовать эту возможность, чтобы получить более полное представление об интерфейсах API и вводе данных, необходимых для успешного прохождения проверки клиентами. Поддержка существующих интерфейсов API Центра партнеров будет прекращена до окончания второго квартала 2021 финансового года. К этому времени вам необходимо будет перейти на новые интерфейсы API Центра партнеров.

## <a name="update---csp-o365-usage-apis-and-csp-license-usage-reports"></a><a name="13"></a>Обновление. API-интерфейсы использования Office 365 для CSP и отчеты об использовании лицензий CSP

- Дата: 29.09.2020
- Характеристики

### <a name="impacted-audience"></a>Затронутая аудитория

Все партнеры, заключающие сделки в рамках программы "Поставщик облачных решений" (CSP) в Центре партнеров. 

### <a name="summary"></a>Итоги 

Определения метрик, предоставляемых в API-интерфейсах использования Office 365 в рамках CSP, будут обновлены, чтобы партнерам было проще составить представление о том, как они используют Office 365. 

### <a name="details"></a>Подробнее

Это дополнение к [объявлению о Центре партнеров](#8), которое было опубликовано 17 сентября 2020 г. Мы вносим изменения, чтобы обеспечить обратную совместимость обновлений API, благодаря чему вам не потребуется вносить какие-либо изменения в интеграцию. 

Как было объявлено ранее, с 30 сентября 2020 г. для метрик использования Office 365 в рамках CSP будет проводиться обновление. 

Оно будет включать изменения определений метрик, доступных в следующих API использования в рамках CSP: 

1. [API сведений об использовании лицензий](/partner-center/develop/get-licenses-usage-information);

2. [API сведений об использовании лицензий клиентами](/partner-center/develop/get-customer-licenses-usage-information);

3.  [API сведений об использовании лицензий партнерами](/partner-center/develop/get-partner-licenses-usage-information).

Обновленные определения приведены ниже. Так как сама схема API изменена не будет, обновлять интеграцию не потребуется. 

| **Метрика** | **Обновленные определения** |
|---------|:---------|
|licensesActive |Число лицензий, использованных за последние 28 дней. |
|licensesQualified |Число пробных и полных лицензий, доступных для SKU продукта. |
|UsagePercent |Активные лицензии в процентах от соответствующих требованиям лицензий. |
|ProratedLicenseUsagePercent|Активные лицензии в процентах от соответствующих требованиям лицензий (агрегированное значение для партнера).  |

## <a name="upcoming-russian-federal-law-vat-changes"></a><a name="12"></a>Предстоящие изменения в федеральном налоговом законодательстве России

### <a name="categories"></a>Категории

- Дата: 25.09.2020
- Характеристики

### <a name="impacted-audience"></a>Затронутая аудитория

Партнеры из России, предоставляющие услуги корпоративного лицензирования (VL), бизнес-решения Майкрософт (MBS), и поставщики облачных решений (CSP).

### <a name="summary"></a>Сводка 

Изменения налогового законодательства России по федеральному закону № 265-ФЗ вступят в силу с 1 января 2021 г.

### <a name="details"></a>Подробнее

Федеральный закон России № 265-ФЗ от 31 июля 2020 г. вносит изменения в подраздел 26 раздела 2 статьи 149 Налогового кодекса России. В соответствии с этими изменениями с 1 января 2021 г. предоставление эксклюзивных прав интеллектуальной собственности на программное обеспечение и электронные базы данных, которые **не включены** в общий **российский** реестр программного обеспечения и электронных баз данных, а также предоставление прав на использование такого программного обеспечения и баз данных (включая исправления и обновления) и предоставление удаленного доступа к ним через Интернет **более не освобождается от НДС**. 

**Учтите, что в связи с приведенными выше изменениями ко всем нематериальным продуктам Майкрософт в России компания MIOL добавит НДС, определяемый применимым законодательством.**

## <a name="enhanced-incentives-support-experience-and-ticket-creation-process"></a><a name="4"></a>Расширенная поддержка программы поощрения и процесс создания запросов

### <a name="details"></a>Сведения

В программу поощрения для партнеров внесены усовершенствования. Они расширяют общие возможности поддержки благодаря улучшенному механизму обнаружения материалов для самостоятельного решения вопросов, а также упрощенному процессу отправки запросов, что ускоряет прямой и обратный сбор данных в процессе поддержки.

#### <a name="new-capabilities"></a>Новые возможности

- Новый интерфейс для самостоятельного решения вопросов на основе механизма поиска доступен в [Центре партнеров](https://aka.ms/IncentivesSupport) и в [Microsoft Partner Network](https://partner.microsoft.com/support/?stage=1) для учетных записей, не принадлежащих Центру партнеров. - Передайте эти сведения другим сотрудникам в своей организации. 
- Используйте новые возможности поддержки в [Центре партнеров](https://aka.ms/IncentivesSupport) или в [Microsoft Partner Network](https://partner.microsoft.com/support/?stage=1) при любой потребности в поддержке в рамках программы поощрения. 

**Примечание**. Запросы на поддержку не будут приниматься в рамках рассылки по электронной почте, так как поддержка этой функции прекращена.
