## <a name="new-skus-bring-the-power-of-project-cortex-and-more-to-customers"></a><a name="5"></a>Новые номера SKU предоставляют клиентам преимущества Project Cortex и многие другие возможности

### <a name="categories"></a>Категории

- Дата: 08.09.2020
- Предложения и рынки

### <a name="impacted-audience"></a>Затронутая аудитория

Все партнеры, заключающие сделки в рамках программы "Поставщик облачных решений" (CSP).

### <a name="summary"></a>Сводка

Для Transact с 1 октября 2020 г. будут доступны новые номера SKU, которые помогут клиентам воспользоваться возможностями [Project Cortex](https://resources.techcommunity.microsoft.com/project-cortex-microsoft-365/) и [Project Turing](https://msturing.org/).

### <a name="details"></a>Сведения

Мы рады сообщить, что с 1 октября 2020 г. будут доступны новые номера SKU, включая [Project Cortex](https://resources.techcommunity.microsoft.com/project-cortex-microsoft-365/) (интеллектуальные службы содержимого) и службу поиска Майкрософт (в том числе соединители Microsoft Graph для поиска), которые предоставят пользователям более эффективные возможности поиска и администрирования данных.  

#### <a name="intelligent-content-services"></a>Интеллектуальные службы содержимого
  
Интеллектуальные службы содержимого используют расширенные возможности ИИ для автоматизации записи, добавления тегов и управления содержимым. Решение создано на основе SharePoint, что дает возможности обработки больших объемов содержимого, извлечения метаданных для автоматизации процессов и высокоточных механизмов обеспечения безопасности и соответствия. Интеллектуальные службы содержимого повышают уровень контроля знаний и расширяют возможности обнаружения, отправляя более точное и достоверное содержимое в сеть знаний. Новые номера SKU интеллектуальных служб содержимого будут доступны в **прейскуранте от 1 октября** (и в предварительном прейскуранте от 1 сентября). 
 
**Примечание**. Интеллектуальные службы содержимого — это имя заполнителя для этого продукта, для которого, вероятнее всего, после 1 октября 2020 г. будет выполнен ребрендинг.
 
**Соединители Microsoft Graph для службы поиска Майкрософт.** Соединители Microsoft Graph позволяют клиентам индексировать элементы во внешних репозиториях для включения в результаты поиска Майкрософт. 

С 1 октября 2020 г.: 

- Microsoft 365 E5 и Office 365 E5 будут предоставлять возможность индексировать до 500 элементов с помощью соединителей Microsoft Graph для службы поиска Майкрософт (не входит в A5).  
- Любой пользователь с набором лицензий или отдельной лицензией, включающей в себя план SharePoint или OneDrive, сможет просматривать результаты поиска с помощью соединителей Microsoft Graph для службы поиска Майкрософт. 
- Будет доступен новый **номер SKU дополнительной емкости соединителя Graph**, который позволит индексировать до 1 млн дополнительных элементов.

## <a name="enhanced-incentives-support-experience-and-ticket-creation-process"></a><a name="4"></a>Расширенная поддержка программы поощрения и процесс создания запросов

### <a name="categories"></a>Категории

- Дата: 02.09.2020
- Характеристики

### <a name="impacted-audience"></a>Затронутая аудитория

Все партнеры.

### <a name="summary"></a>Итоги

В интерфейсе поддержки программы поощрения для партнеров расширены возможности обнаружения материалов для самостоятельного решения вопросов и упрощена отправка интерактивных запросов.

### <a name="details"></a>Сведения

В программу поощрения для партнеров внесены усовершенствования. Они расширяют общие возможности поддержки благодаря улучшенному механизму обнаружения материалов для самостоятельного решения вопросов, а также упрощенному процессу отправки запросов, что ускоряет прямой и обратный сбор данных в процессе поддержки.

#### <a name="new-capabilities"></a>Новые возможности

- Новый интерфейс для самостоятельного решения вопросов на основе механизма поиска доступен в [Центре партнеров](https://aka.ms/IncentivesSupport) и в [Microsoft Partner Network](https://partner.microsoft.com/support/?stage=1) для учетных записей, не принадлежащих Центру партнеров. Новый интерфейс содержит новые инструкции по устранению неполадок, раздел с часто задаваемыми вопросами и ссылки на статьи для самостоятельного решения вопросов, так же как и страница поддержки Microsoft Partner Network.
- Инициируйте новый запрос в службу поддержки непосредственно в Центре партнеров. Если после просмотра доступной справки, относящейся к конкретной теме, вам по-прежнему нужна помощь, вы можете создать запрос только непосредственно в Центре партнеров.
- Для каждой статьи для самостоятельного решения вопросов доступны новые уточняющие вопросы о проблеме. Ответив на эти вопросы, вы предоставите агенту множество важных подробных сведений, что значительно сократит время, требующееся для обмена данными при подаче запроса в службу поддержки.


### <a name="next-steps"></a>Дальнейшие действия

- Передайте эти сведения другим сотрудникам в своей организации. 
- Используйте новые возможности поддержки в [Центре партнеров](https://aka.ms/IncentivesSupport) или в [Microsoft Partner Network](https://partner.microsoft.com/support/?stage=1) при любой потребности в поддержке в рамках программы поощрения. 

**Примечание**. Запросы на поддержку не будут приниматься в рамках рассылки по электронной почте, так как поддержка этой функции прекращена.

## <a name="upcoming-new-microsoft-dynamics-365-offers"></a><a name="3"></a>Планирующиеся к выпуску новые предложения Microsoft Dynamics 365

### <a name="categories"></a>Категории

- Дата: 01.09.2020
- Предложения и рынки

### <a name="impacted-audience"></a>Затронутая аудитория

Партнеры, которые ведут деятельность с Microsoft Dynamics 365

### <a name="summary"></a>Сводка

Планирующиеся к выпуску предложения для клиентов Microsoft Dynamics 365 включают выпуск Microsoft Project Operations и дополнительные среды для Business Central.

### <a name="details"></a>Сведения

1 октября 2020 г. будут доступны указанные ниже новые предложения.

#### <a name="project-operations"></a>Операции проекта

1 октября 2020 г. корпорация Майкрософт выпустит предложения Dynamics 365 Project Operations, и удалит Dynamics 365 Project Service Automation (PSA) для программы "Поставщик облачных решений" (CSP). Новым клиентам следует приобретать Dynamics 365 Project Operations. 

Для существующих клиентов Dynamics 365, использующих PSA:

- Продолжайте использовать Dynamics 365 PSA до 1 октября 2020 г. или до окончания срока действия последнего контракта или продления в зависимости от того, что наступит раньше.
- Dynamics 365 PSA будет автоматически обновляться в рамках CSP на один год до 1 октября 2021 г., что позволяет использовать контракты до 1 октября 2022 г.
- До 31 декабря 2020 г. в текущие лицензии на Dynamics 365 PSA будут автоматически добавлены предложения Project Operations.
- По окончании срока действия контрактов для Dynamics 365 PSA клиентам нужно будет перейти на предложения Project Operations, у которых будет нулевой минимальный пользовательский номер SKU для перехода с PSA.

Сведения об идентификаторе предложения см. на вкладке Project Operations в [документе по предложениям Dynamics CSP за октябрь 2020 г.](https://partner.microsoft.com/resources/detail/microsoft-dynamics-365-new-ids-offers-csp-october-2020-xls)

#### <a name="additional-resources"></a>Дополнительные ресурсы

- [Ресурсы перехода](https://partner.microsoft.com/asset/collection/psa-transition-resources#/) для PSA
- Блог [Знакомство с Microsoft Dynamics 365 Project Service Automation](https://cloudblogs.microsoft.com/dynamics365/bdm/2020/02/20/introducing-microsoft-dynamics-365-project-operations/)
- Учебные материалы по [возможностям в составе Microsoft Dynamics 365 Project Service Automation](https://assetsprod.microsoft.com/mpn/what-is-coming-in-d365-project-service-automation-dyn767pal.pdf)

#### <a name="expansion-of-business-central-offers-into-additional-countries"></a>Доступность предложений Business Central в других странах

1 октября 2020 г. корпорация Майкрософт начинает предоставлять предложения для облака Business Central в Бразилии, Ирландии, Литве, на Фарерских островах и в Гренландии.

#### <a name="business-central-additional-environments"></a>Дополнительные среды Business Central

1 октября 2020 г. станут доступны новые предложения надстроек для облака Business Central. Они позволят клиентам добавлять дополнительные среды. 

Сведения об идентификаторе предложения см. на вкладке **дополнительных сред** в документе по [предложениям Dynamics CSP за октябрь 2020 г.](https://partner.microsoft.com/resources/detail/microsoft-dynamics-365-new-ids-offers-csp-october-2020-xls)

## <a name="office-365-calling-plan-will-be-available-in-new-markets-on-october-1-2020"></a><a name="2"></a>Тарифный план Office 365 станет доступным на новых рынках 1 октября 2020 г.

### <a name="categories"></a>Категории

- Дата: 01.09.2020
- Предложения и рынки

### <a name="impacted-audience"></a>Затронутая аудитория

Партнеры в рамках программы "Поставщик облачных решений" (CSP), прямые и косвенные поставщики, а также косвенные торговые посредники

### <a name="summary"></a>Сводка

В Майкрософт рады объявить о том, что 1 октября 2020 г. тарифный план станет доступным на следующих новых рынках: Австрия, Дания, Италия, Португалия, Швейцария и Швеция. 

### <a name="details"></a>Сведения

1 октября 2020 г. тарифный план Microsoft 365 станет доступным на следующих новых рынках: Австрия, Дания, Италия, Португалия, Швейцария и Швеция. Поддержка новых географических областей будет доступна для коммерческого, некоммерческого и международного государственного секторов, а также для облака сообщества для государственных организаций в рамках Соглашения Enterprise (EA) или Соглашения Enterprise Subscription (EAS), Соглашения о регистрации для решений для образования (EES), Web Direct и CSP. 

Все функции тарифного плана будут доступны для планов внутренних и международных звонков от Майкрософт, включая планы внутренних звонков на 120, 240 и 1200 минут на пользователя в месяц (p/u/m), а также международный план с 1200 минутами для внутренних звонков p/u/m и с 600 минутами для международных звонков p/u/m. 

Дополнительные сведения о тарифных планах, чрезмерном использовании и кредитах на связь см. в разделе о тарифном плане на странице с [вопросами и ответами о современных рабочих лицензиях](https://partner.microsoft.com/resources/collection/partner-center-edu-validation-enhancements).

Сейчас тарифный план доступен в Бельгии, Германии, Ирландии, Испании, Канаде, Нидерландах, Пуэрто-Рико (каналы доступны только в рамках EA и EES), Соединенном Королевстве, США и Франции (в Австралии и Японии тарифный план доступен через сторонних поставщиков).
