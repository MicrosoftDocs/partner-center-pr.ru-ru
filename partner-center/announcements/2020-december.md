## <a name="sdk-release-on-net-standard-v1163"></a><a name="4"></a>Выпуск пакета SDK для .NET Standard (версия 1.16.3)

### <a name="categories"></a>Категории

- Дата: 08.12.2020
- Характеристики

### <a name="impacted-audience"></a>Затронутая аудитория

Партнеры с прямым выставлением счетов и непрямые поставщики, которые участвуют в программе CSP и используют пакет SDK Центра партнеров для .NET.

### <a name="details"></a>Сведения

С 8 декабря 2020 г. партнеры получили возможность скачивать версию пакета SDK [MicrosoftPartnerCenter.NETSDK (коллекция NuGet | Microsoft.Store.PartnerCenter 1.16.3)](https://www.nuget.org/packages/Microsoft.Store.PartnerCenter/1.16.3), а также обновленные общедоступные [примеры для пакетов SDK Центра партнеров на GitHub](https://github.com/Microsoft/Partner-Center-DotNet-Samples). Эта версия включает перечисленные ниже обновления.
 
**SelfServePolicies — добавлены новые функции:**

- [GetSelfServePolicies](/partner-center/develop/get-a-self-serve-policy-by-id.md);
- [GetListOfSelfServicePolicies](/partner-center/develop/get-a-list-of-self-serve-policies.md);
- [CreateSelfServePolicies](/partner-center/develop/create-a-self-serve-policy.md);
- [UpdateSelfServePolicies](/partner-center/develop/update-a-self-serve-policy.md);
- [DeleteSelfServePolicies](/partner-center/develop/delete-a-self-serve-policy.md).
 
**Профиль компании клиента:**

- Добавлено поле [OrganizationRegistrationNumber](/partner-center/develop/create-a-customer.md).
 
**CustomerBillingProfile.DefaultAddress:**

- Добавлено поле MiddleName.
 
### <a name="next-steps"></a>Next Steps

- Скачайте последнюю версию [MicrosoftPartnerCenter.NETSDK (коллекция NuGet | Microsoft.Store.PartnerCenter 1.16.3)](https://www.nuget.org/packages/Microsoft.Store.PartnerCenter/1.16.3).

## <a name="an-update-of-the-us-microsoft-365-business-voice-with-calling-plan-offer-is-coming-soon"></a><a name="2"></a>Обновление предложения "Голосовая связь Microsoft 365 бизнес с тарифным планом" для США ожидается в ближайшее время

### <a name="categories"></a>Категории

- Дата: 03.12.2020
- Предложения и рынки

### <a name="summary"></a>Сводка 

С 1 января 2021 г. корпорация Майкрософт начнет перевод партнеров и клиентов из США на новое предложение "Голосовая связь Microsoft 365 бизнес с тарифным планом". Никаких действий со стороны партнеров не требуется.

### <a name="impacted-audience"></a>Затронутая аудитория 

Все партнеры, заключающие сделки в рамках программы "Поставщик облачных решений" (CSP).

### <a name="details"></a>Сведения

Майкрософт заменит существующее предложение "Голосовая связь Microsoft 365 бизнес с тарифным планом" для США новым предложением в соответствии с внутренними требованиями к телекоммуникационным продуктам. Новое предложение будет иметь тот же набор возможностей по таким же ценам. Изменения в предложении являются внутренними для Майкрософт и не должны повлиять на то, как партнеры по программе CSP продвигают, продают или поддерживают предложение "Голосовая связь бизнес с тарифным планом". Изменение касается только этого предложения.

Для новых продаж новое предложение следует использовать с 1 января 2021 г. Прежнее предложение будет заменено в прейскуранте новым предложением.

Существующие клиенты будут автоматически переведены на новое предложение, когда для них наступит дата продления. Никаких действий от партнеров, а также клиентов, продлевающих срок действия продукта, не требуется.

## <a name="now-live-partner-center-api-updates-and-user-interface-enhancements-for-the-education-customer-validation-process"></a><a name="1"></a>Уже доступно. Обновления API Центра партнеров и улучшения пользовательского интерфейса для проверки клиентов из сферы образования

### <a name="categories"></a>Категории

- Дата: 03.12.2020
- Возможности

### <a name="impacted-audience"></a>Затронутая аудитория 

Партнеры, продающие предложения для образовательных учреждений в рамках программы "Поставщик облачных решений" (CSP).

### <a name="summary"></a>Сводка 

Корпорация Майкрософт работает на основе принципов доверия, Мы стремимся предоставлять соответствующий требованиям, безопасный и защищенный метод проверки клиентов для работы с предложениями для образовательных учреждений по программе "Поставщик облачных решений". В рамках этого обязательства мы представляем улучшения API Центра партнеров и пользовательского интерфейса. Они повысят наглядность и расширят возможности визуального контроля при проверке клиентов, а также позволят вводить более точные данные, что повысит вероятность успешного прохождения проверки клиентами. 

**Улучшения Центра партнеров** 

- Новые интерфейсы API квалификаций GET и POST для поддержки точной записи данных и улучшения процесса проверки клиентов корпорацией Майкрософт. 

- Улучшения пользовательского интерфейса для поддержки точной записи данных и оптимизации процесса проверки клиентов корпорацией Майкрософт. 

**Объявление существующих API GET и PUT для проверки соответствия требованиям нерекомендуемыми** 

**К концу февраля 2021 года** поддержка существующих API GET и PUT для проверки соответствия требованиям будет прекращена. В это время вам нужно будет перейти на новые API Центра партнеров GET и POST, чтобы иметь возможность приобретать предложения для образовательных учреждений.  

Тестирование. 

Чтобы получить более полное представление об интерфейсах API и вводе данных, необходимых для успешной проверки клиентов, партнеры смогут протестировать эти улучшения. Партнеры, которые хотят принять участие в тестировании, должны скачать [руководство по тестированию клиентов из сферы образования в Центре партнеров](https://partner.microsoft.com/resources/detail/partner-center-edu-testing-guide-pdf) с указаниями о подготовке и регистрации и описанием этапа тестирования.

**Библиотеки и музеи** 

Мы рады сообщить о том, что в дополнение к описанным выше улучшениям мы предоставим предложения для библиотек и музеев по ценам для образовательных учреждений. Это расширит базу клиентов из образовательного сектора, которым можно предоставлять предложения в рамках CSP. 

Корпорация Майкрософт оставляет за собой право просматривать сведения о состоянии любого клиента или предлагаемого клиента на предмет его соответствия требованиям для работы с предложениями для образования. См. подробные сведения о [требованиях, которым должен соответствовать пользователь, для работы с предложениями для образовательных учреждений](https://www.microsoftvolumelicensing.com/DocumentSearch.aspx?Mode=3&DocumentTypeId=7).
