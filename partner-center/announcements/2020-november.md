## <a name="new-microsoft-teams-advanced-communications-skus-for-government-community-cloud-gcc-coming-soon"></a><a name="4"></a>В ближайшее время станут доступны новые номера SKU Microsoft Teams Advanced Communications, предназначенные для облака сообщества для государственных организаций (GCC)

### <a name="categories"></a>Категории

- Дата: 02.11.2020
- Предложения и рынки

### <a name="summary"></a>Итоги

С 1 декабря 2020 года станут доступны новые номера SKU надстройки расширенного взаимодействия для Microsoft Teams в GCC.

### <a name="impacted-audience"></a>Затронутая аудитория

Все партнеры, заключающие сделки в рамках программы "Поставщик облачных решений" (CSP).

### <a name="details"></a>Сведения

Доступна новая надстройка расширенного взаимодействия для Microsoft Teams в GCC. Стоимость — 12 долл. США на пользователя в месяц. Номера SKU надстройки можно приобрести вдобавок к любому другому набору Microsoft 365, содержащему Microsoft Teams. Надстройка расширенного взаимодействия — это новый набор возможностей для больших собраний, политик взаимодействия и интеграции, а также дополнительные средства для ИТ-управления. 

#### <a name="offer-details"></a>Подробности предложения

   |**Название предложения**|**Идентификатор предложения**|**Идентификатор материалов**|
   |-------------------|:------|:------|
   |Расширенное взаимодействие для облака сообщества для государственных организаций|56fe76f5-f4ba-4fac-9561-d0daf59b01a1|7FB-00003|

#### <a name="frequently-asked-questions"></a>Часто задаваемые вопросы 

**Что собой представляет надстройка расширенного взаимодействия?** Эта новая надстройка Microsoft Teams предоставляет клиентам еще больше возможностей для взаимодействия. Ее можно приобрести вместе с любым номером SKU Microsoft 365, на который оформлена подписка.

**Сколько оно стоит?** Коммерческая цена ERP — 12 долл. США на пользователя в месяц.

**Какие клиенты смогут приобрести надстройку?** Эту надстройку могут приобрести клиенты GCC.

**Как можно будет приобрести предложение?** Надстройку можно приобрести в рамках Соглашения Enterprise, подписки с Соглашением Enterprise, программы Enrollment for Education Solutions, CSP или через Web Direct.

**Где можно продавать это предложение?** Его можно продавать на рынках США.

**Каковы предварительные требования?** Эту надстройку можно предлагать вместе с любым пакетом Microsoft 365 или Office 365, включающим Microsoft Teams.

### <a name="next-steps"></a>Дальнейшие шаги

Передайте эти сведения соответствующим контактным лицами в вашей организации, а также узнайте о возможностях дополнительных и перекрестных продаж. Ознакомьтесь с разделом ресурсов в [руководстве по Teams для партнеров](https://aka.ms/teamscallingmeetingsguide).

## <a name="introducing-microsoft-365-business-voice-for-nonprofit-organizations"></a><a name="2"></a>Представляем Голосовую связь Microsoft 365 бизнес для некоммерческих организаций

### <a name="categories"></a>Категории

- Дата: 02.11.2020
- Предложения и рынки

### <a name="summary"></a>Итоги

1 ноября 2020 г. корпорация Майкрософт представила новые номера SKU для Голосовой связи Microsoft 365 бизнес для некоммерческих организаций.

### <a name="impacted-audience"></a>Затронутая аудитория

Прямые поставщики, косвенные поставщики и косвенные торговые посредники.

### <a name="details"></a>Сведения

1 ноября 2020 г. корпорация Майкрософт представила новые номера SKU для Голосовой связи для бизнеса. Полный пакет доступен в Канаде, Соединенном Королевстве и США. Голосовая связь для бизнеса (без тарифного плана) будет доступна на всех остальных рынках. 

Голосовая связь Microsoft 365 бизнес — это облачная система телефонии для малых и средних предприятий, встроенная в Office 365. Включив эту систему в подписку Office 365 клиента, вы получите комплексное решение для коммуникации и совместной работы с поддержкой звонков, чатов и собраний в едином приложении — Microsoft Teams.

Сведения о ценах см. в прейскуранте.

Решение "Голосовая связь Microsoft 365 бизнес" можно добавить в следующие подписки не более чем с 300 пользователями.

- Office 365: бизнес базовый, бизнес премиум, A1, E1, A3 и E3.
- Microsoft 365: бизнес, A3 и E3.

## <a name="cloud-solution-provider-csp-promo-for-microsoft-365-business-voice-is-now-available"></a><a name="1"></a>Акционное предложение в рамках программы "Поставщик облачных решений" (CSP) для Голосовой связи Microsoft 365 бизнес уже доступно

### <a name="categories"></a>Категории

- Дата: 02.11.2020
- Предложения и рынки

### <a name="summary"></a>Итоги

Доступны скидки на новые и продленные подписки на Голосовую связь Microsoft 365 бизнес с тарифным планом и Голосовую связь Microsoft 365 бизнес без тарифного плана.

Все партнеры, работающие по программе CSP.

В течение 12 месяцев на Голосовую связь Microsoft 365 бизнес с тарифным планом и Голосовую связь Microsoft 365 бизнес без тарифного плана будут действовать скидки в размере 25 % и 33 % соответственно.
