## <a name="now-available-windows-7-year-2-esu-for-cloud-solution-provider-csp"></a><a name="6"></a>Теперь для партнеров CSP доступны дополнительные обновления для системы безопасности Windows 7 на второй год

### <a name="categories"></a>Категории

- Дата: 29.06.2020
- Введите команду: Предложения и рынки
- Состояние Уже доступно

### <a name="summary"></a>Сводка

Теперь в [коллекции ресурсов](https://partner.microsoft.com/resources/collection/windows-esu-csp#/) доступны цены на дополнительные обновления для системы безопасности Windows 7 на второй год.

### <a name="impacted-audience"></a>Затронутая аудитория

Все партнеры, участвующие в программе "Поставщик облачных решений" (CSP).

### <a name="details"></a>Подробности

Поддержка Windows 7 прекратилась 14 января 2020 г. В то время как многие клиенты значительно продвинулись на пути к развертыванию Windows 10, в корпорации Майкрософт понимают, что разные пользователи находятся на разных этапах процесса обновления систем. Период действия дополнительных обновлений для системы безопасности Windows 7 на первый год заканчивается 12 января 2021 г. Мы знаем, что клиенты начинают строить планы на следующий год.  

Для клиентов, которые хотят и дальше использовать устройства с Windows 7 в 2021 году, мы предлагаем возможность приобрести дополнительные обновления для системы безопасности Windows 7 на второй год у партнеров CSP. Это поможет клиентам принять решение относительно перехода на Windows 10. Дополнительные сведения см. в [коллекции ресурсов](https://partner.microsoft.com/resources/collection/windows-esu-csp#/).

### <a name="next-steps"></a>Дальнейшие действия
Передайте эти сведения соответствующим контактным лицам в своей организации.

## <a name="partner-center-relationship-removal-feature-update"></a><a name="5"></a>Обновление функции удаления связей в Центре партнеров

### <a name="categories"></a>Категории

- Дата: 29.06.2020
- Введите команду: Характеристики
- Состояние Уже доступно
- 
### <a name="summary"></a>Сводка

С 29 июня 2020 г. непрямые поставщики и непрямые торговые посредники по программе CSP могут удалять нисходящие взаимоотношения с помощью Центра партнеров, не отправляя запрос в службу поддержки Центра партнеров. Это сэкономит партнерам значительное количество времени, которое требовалось на выполнение этих действий.

### <a name="impacted-audience"></a>Затронутая аудитория

Прямые и непрямые поставщики облачных решений в рамках программы "Поставщик облачных решений" (CSP).

### <a name="scenarios-supported"></a>Поддерживаемые сценарии:

- Непрямые поставщики, которые установили связь с непрямым торговым посредником, могут удалить эту связь с помощью Центра партнеров.
- Непрямые торговые посредники, которые установили связь с конечным клиентом, могут удалить эту связь с помощью Центра партнеров.

Действия в приведенных выше сценариях следует выполнять агенту по администрированию Центра партнеров.

### <a name="next-steps"></a>Дальнейшие действия

- Дополнительные сведения см. в [коллекции ресурсов](https://partner.microsoft.com/resources/collection/partner-center-relationship-removal#).
- Ознакомьтесь с материалами из коллекции по подготовке к работе: Обновление функции удаления связей в Центре партнеров.
- Передайте эти сведения сотрудникам своей организации, чтобы соответствующая команда могла проверить процессы и внести в них необходимые изменения.

## <a name="billing-frequency-default-changing-from-annual-to-monthly"></a><a name="4"></a>Периодичность выставления счетов по умолчанию изменяется с ежегодной на ежемесячную

### <a name="categories"></a>Категории

- Дата: 17.06.2020
- Введите команду: Характеристики
- Состояние Ожидается в ближайшее время

### <a name="summary"></a>Сводка

Периодичность выставления счетов по умолчанию в интерфейсе покупки Центра партнеров изменяется с ежегодной на ежемесячную.

### <a name="impacted-audience"></a>Затронутая аудитория

Прямые и непрямые поставщики облачных решений (CSP).

### <a name="details"></a>Подробности

Параметр периодичности выставления счетов определяет, как часто партнеру выставляются счета за приобретенные службы. Параметр отображается на шаге **проверки**, когда партнер осуществляет покупку в каталоге Центра партнеров. До сегодняшнего дня этот параметр по умолчанию имел значение **Ежегодно**, так как для служб на основе лицензий было введено ежегодное выставление счетов. На основе отзывов партнеров с 17 июня команда Центра партнеров меняет это значение по умолчанию с **Ежегодно** на **Ежемесячно**, чтобы он лучше соответствовал ожиданиям партнеров. Партнеры могут изменить это значение перед отправкой транзакции на странице проверки или [после завершения транзакции на странице управления подпиской](../billing-annual-monthly.md#change-billing-frequency). Учтите, что для предложений, поддерживающих только ежегодное выставление счетов, по умолчанию будет продолжать указываться соответствующее значение. Это изменение значения по умолчанию касается только пользовательского интерфейса портала Центра партнеров и не влияет на работу API-интерфейсов Центра партнеров.

Мы надеемся, что это изменение снизит вероятность случайного приобретения партнерами подписок с ежегодным выставлением счетов.

## <a name="changes-to-microsoft-365-f1"></a><a name="3"></a>Изменения в Microsoft 365 F1

- Состояние Уже доступно

В Microsoft 365 F1 будут добавлены новые службы. 15 июня будет выпущено новое предложение, которое заменит предыдущее предложение F1.

   |Microsoft 365 F1 (пробная версия)|dd97e36a-f0df-4a9a-b461-625420656511|1PI-00006|M365_F1|

### <a name="call-to-action"></a>Призыв к действию
Партнеры, которые уже приобрели текущий продукт Microsoft 365 F1, не должны выполнять никаких действий. Планы обслуживания для базовой подписки на Exchange и Поиска (Майкрософт) будут заполнены до конца июня согласно существующим планам обслуживания.

Партнерам, которые хотят приобрести Microsoft 365 F1 после 15 июня, следует изучить информацию о новых предложениях F1, чтобы убедиться, что они приобретают нужное предложение. Новые идентификаторы предложений F1 и другие сведения будут указаны в июльском прейскуранте для партнеров CSP, который выйдет 1 июля.
