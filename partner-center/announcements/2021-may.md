## <a name="default-bank-profile-functionality-is-available-in-partner-center-for-incentive-enrollment"></a><a name="7"></a>В Центре партнеров доступны функции настройки банковского профиля по умолчанию для регистрации в программе поощрений

### <a name="categories"></a>Категории

- Дата: 07.05.2021 г.
- Характеристики

### <a name="impacted-audience"></a>Затронутая аудитория

Администраторы программ поощрений

### <a name="details"></a>Сведения

Мы рады сообщить о том, что в Центре партнеров теперь доступны функции настройки банковского профиля по умолчанию. С помощью этих функций администраторы программ поощрения могут настраивать банковское профили по умолчанию и назначать их расположениям Microsoft Partner Network (MPN). После настройки эти профили по умолчанию будут использоваться корпорацией Майкрософт, когда партнер захочет зарегистрироваться в какой-либо программе поощрения для этого места MPN.

Подробные инструкции см. в статье [Настройка банковского профиля по умолчанию](https://docs.microsoft.com/partner-center/incentives-create-and-manage-your-payout-and-tax-profiles#set-up-a-default-bank-profile).

#### <a name="enrolling-in-an-incentive-program-when-a-default-bank-profile-has-been-assigned"></a>Регистрация в программе поощрения с назначенным банковским профилем по умолчанию

Если вы пытаетесь зарегистрировать расположение MPN в программе поощрения и для него уже установлен профиль по умолчанию, вы увидите этот профиль при запуске регистрации и принятии приглашения. Вы также увидите доступные налоговые профили, созданные для этого расположения MPN. Если у Майкрософт есть все необходимые сведения банковского и налогового профилей, вам будет предложено нажать кнопку "Отправить", чтобы завершить регистрацию.

Вы также сможете выбрать банковский профиль, отличный от профиля по умолчанию. Если корпорации Майкрософт потребуются дополнительные сведения о платежном или налоговом профиле или валюте, вам будет предложено нажать "Продолжить", после чего откроется страница соответствующего профиля, где вы сможете указать недостающие сведения.

Регистрация считается завершенной только после того, как корпорация Майкрософт проверила платежный и налоговый профили, предоставленные для регистрации. Дополнительные сведения о платежных и налоговых профилях см. [здесь](https://docs.microsoft.com/partner-center/incentives-create-and-manage-your-payout-and-tax-profiles).

## <a name="csp-perpetual-software-coming-soon-to-china"></a><a name="5"></a>Программное обеспечение с бессрочной лицензией в рамках CSP скоро станет доступно в Китае.

### <a name="categories"></a>Категории

- Дата: 05.05.2021
- Характеристики

### <a name="summary"></a>Сводка

Начиная с первого квартала финансового года (FY22 Q1) в рамках программы CSP компании Microsoft China Corporation Limited (MCCL) будет доступно программное обеспечение с бессрочной лицензией.

### <a name="impacted-audience"></a>Затронутая аудитория

Партнеры в рамках программы CSP MCCL

### <a name="details"></a>Сведения

Количество участников программы CSP растет в геометрической прогрессии и сейчас составляет более 90 000 компаний. Это наша основная инициатива партнерских продаж для небольших и средних компаний, а также для небольших корпоративных клиентов по новой коммерческой модели. Партнеры могут создавать и предлагать дифференцированные предложения клиентам, продавая продукты категории Microsoft Cloud и сторонние решения с коммерческой платформы Майкрософт наряду с собственными решениями и службами. Они могут управлять всем жизненным циклом взаимодействия с клиентами, зарабатывая репутацию надежных консультантов.

#### <a name="expanding-opportunities-for-partners"></a>Растущие возможности для партнеров

В свете этого корпорация Майкрософт расширяет возможности для партнеров CSP, выпуская бессрочные коммерческие лицензии на программное обеспечение, не требующие участия в программе Software Assurance (SA), для локальных развертываний в рамках программы CSP MCCL*, начиная с 1 квартала 2022 финансового года Майкрософт. Это обеспечит клиентам дополнительную гибкость для лицензирования при цифровой трансформации.

Как уже [было объявлено ранее](https://blogs.partner.microsoft.com/mpn/expanding-opportunities-for-partners-in-the-cloud-solution-provider-program/), начиная с 1 января 2022 г. вы не сможете продлить действие SA или приобрести новые лицензии на программное обеспечение в рамках программы Open License. Новые покупки с лицензиями должны осуществляться только в рамках программы CSP MCCL.

* Это относится только к CSP MCCL. На условия CSP 21Vianet изменение не повлияет.

### <a name="next-steps"></a>Дальнейшие действия

Партнеры, зарегистрированные или планирующие регистрацию в CSP MCCL, должны проанализировать приведенные ниже материалы и предоставить их соответствующим заинтересованным лицам в своей организации.

- [Материалы о готовности CSP и Центра партнеров для региона Большого Китая (GCP)](https://partner.microsoft.com/resources/collection/software-in-csp-cht#/): вводный обзор, руководство по лицензированию, заказ в Центре партнеров, видеоинструкции по API, часто задаваемые вопросы и многое другое. Субтитры доступны на упрощенном или традиционном китайском.

- [Уведомление об изменениях в программе Microsoft Open License в новостях об условиях лицензирования Майкрософт](https://www.microsoft.com/zh-cn/licensing/news/microsoft_open_license_program_changes): дополнительные сведения об изменениях в программе Open License и возможностях лицензирования. См. также связанные материалы об изменениях в программе Open License на [веб-сайте Майкрософт для партнеров](https://partner.microsoft.com/resources/collection/csp-open-evolution-to-a-better-experience#/).

## <a name="final-reminder-deprecation-of-get-qualification-on-may-6-2021"></a><a name="4"></a>Последнее напоминание: поддержка квалификации GET будет прекращена 6 мая 2021 г.

### <a name="categories"></a>Категории

- Дата: 04.05.2021

- Характеристики

### <a name="impacted-audience"></a>Затронутая аудитория

Партнеры, продающие предложения для образовательных учреждений, некоммерческих организаций и предложения облака сообщества для государственных организаций (GCC) в рамках программы "Поставщик облачных решений" и использующие API Центра партнеров.

### <a name="details"></a>Сведения

Это объявление дополняет сведения об [усовершенствованиях в Центре партнеров, выпущенных в декабре](https://docs.microsoft.com/partner-center/announcements/2020-december#1). Новые API-интерфейсы POST позволяют приобретать предложения для образовательных учреждений, а новые API-интерфейсы GET — специальные предложения для некоммерческих организаций и GCC.

## <a name="corrections-to-may-1-2021-license-based-services-price-list"></a><a name="3"></a>Исправления к прейскуранту служб на основе лицензий за 1 мая 2021 г.

### <a name="categories"></a>Категории

- Дата: 04.05.2021

- Предложения и рынки

### <a name="impacted-audience"></a>Затронутая аудитория

Косвенные поставщики и партнеры с прямым выставлением счетов, которые предоставляют службы на основе лицензий в рамках программы "Поставщик облачных решений".

### <a name="details"></a>Сведения

Прейскурант для служб на основе лицензий, опубликованных на 1 мая 2021 г., содержал неправильную цену в USD для Microsoft 365 бизнес стандартный. Опубликованная ориентировочная цена составляла 8 долларов, хотя должна была быть 10 долларов США. Прейскурант для служб на основе лицензий был обновлен 3 марта 2021 г. в 11:30 PST с соответствующими изменениями. Такие исправления применялись только к следующему:

- Название предложения: Microsoft 365 бизнес стандартный

- Идентификатор предложения: 031c9e47-4802-4248-838e-778fb1d2cc05

- Валюты: доллар США (USD)

- До изменений: ориентировочная цена 8 долларов США

- После изменений: ориентировочная цена 10 долларов США

Эти изменения касаются только указанного выше предложения. Исправления для других предложений не вносились. Ошибка присутствовала только в прейскуранте; она не затрагивала фактическую цену и не могла повлиять на выставление счетов в течение этого периода.

### <a name="next-steps-and-resources"></a>Дальнейшие действия и ресурсы

- Партнеры, предлагающие службы на основе лицензий, должны скачать актуальную версию прейскуранта.
