[здесь](https://partner.microsoft.com/training/assets/collection/microsoft-contract-requirements-and-compliance-18552#/).

## <a name="windows-7-extended-security-updates-esu-year-2-2021-product-key-issue-resolved"></a><a name="3"></a>Устранена проблема с ключом продукта для Расширенных обновлений безопасности (ESU) Windows 7, год 2 (2021)

### <a name="categories"></a>Категории

- Дата: 02.07.2021
- Характеристики
 
### <a name="impacted-audience"></a>Затронутая аудитория

Все партнеры, которые приобрели Расширенные обновления безопасности (ESU) Windows 7, год 2 (2021), в период с 18 мая 2021 г. по 10 июня 2021 г.

### <a name="details"></a>Сведения

Обнаружена временная проблема, связанная с Расширенными обновлениями безопасности (ESU) Windows 7, год 2 (2021). Для заказов, размещенных в период с 18 мая 2021 г. по 10 июня 2021 г., партнеры/клиенты могли получить ключ продукта для номера SKU года 1 (2020). Эта проблема устранена. Для всех заказов, размещенных после 10 июня 2021 г., будет предоставляться правильный ключ для Расширенных обновлений безопасности (ESU) Windows 7, год 2 (2021). Для заказов, размещенных в указанный период, клиентам необходимо оформить возврат старого заказа и приобрести продукт повторно.

### <a name="next-steps"></a>Дальнейшие шаги

Чтобы начать повторное приобретение Расширенных обновлений безопасности (ESU) Windows 7, год 2 (2021), обратитесь к разделу [Приостановка или отмена подписки](../create-a-new-subscription.md#suspend-or-cancel-a-subscription).

## <a name="dynamics-365-cloud-migration-promotion-and-retirement-of-discounted-offers"></a><a name="1"></a>Рекламная акция Dynamics 365 Cloud Migration и прекращение срока действия предложений со скидками

### <a name="categories"></a>Категории

- Дата: 01.07.2021
- Возможности

### <a name="summary"></a>Сводка

Корпорация Майкрософт рада объявить о запуске рекламной акции Dynamics 365 Cloud Migration 1 августа 2021 г.

### <a name="impacted-audience"></a>Затронутая аудитория

Партнеры по программе "Поставщик облачных решений" (CSP), локальные партнеры Dynamics и LSP-партнеры

### <a name="details"></a>Сведения

Благодаря программе CSP партнеры могут провести цифровую трансформацию организаций своих клиентов с применением ведущих в отрасли облачных технологий. В рамках этой работы мы прекращаем использование существующих номеров SKU со скидками (см. вкладку "Номера SKU, которые перестают использоваться" в рекламной акции Dynamics 365 Cloud Migration и [файл Excel с номерами SKU, которые перестают использоваться](https://partner.microsoft.com/resources/detail/dynamics-365-cloud-promotion-retired-skus-xls)) 31 июля 2021 г. и заменяем эти номера SKU новыми номерами SKU рекламной акции Dynamics 365 Cloud Migration.

Корпорация Майкрософт рада объявить о запуске рекламной акции Dynamics 365 Cloud Migration 1 августа 2021 г. В рамках новой рекламной акции предоставляется скидка на веб-службы Dynamics 365, чтобы способствовать переходу клиентов на облако. Эта акция также поддерживает клиентов, которые начали цифровую трансформацию в рамках устаревших предложений по миграции, помогая им перейти на расценки для новой рекламной акции.

#### <a name="dynamics-365-cloud-migration-promotion"></a>Рекламная акция Dynamics 365 Cloud Migration

**Преимущества**

Рекламная акция Cloud Migration предоставляет клиентам следующие преимущества:  

- скидка 40 % на Dynamics 365 Online Services с подпиской на три года, обеспечивающей предсказуемость цен при миграции в облако;

- права на двойной доступ (одновременное локальное и облачное использование) для поддержки пользователей до прекращения работы локального решения после завершения миграции в облако;

- поддержка поэтапной облачной миграции, позволяющая организациям добавлять новых пользователей в течение срока действия рекламной акции.

**Условия**

Программа лицензирования. Поставщик служб шифрования

Продукт: рекламная акция Dynamics 365 Cloud Migration

Скидка, предоставляемая в рамках рекламной акции: 40 %

Длительность рекламной акции: с 1 августа 2021 г. по 30 июня 2023 г.

Срок действия подписки: подписка на три года; продление по цене рекламной акции невозможно.

Требования к лицензированию.

- Все локальные лицензии Dynamics (Dynamics AX, CRM, лицензии для партнеров, являющихся государственными организациями, NAV, лицензия на подписку и др.) с активным планом Dynamics Customer Services and Support (например, Enhancement Plan или Advantage Plan) или Software Assurance (SA) и продлением для веб-служб Dynamics 365.

- Все веб-службы Dynamics 365 с активными подписками для Dynamics 365 из SA или Dynamics Price List, надстройки Dynamics 365 Cloud, предложения Dynamics 365 Qualified или предложения Dynamics 365 Cloud Migration.

- Эта рекламная акция не применяется к новейшим облачным клиентам или неактивным локальным клиентам Dynamics. (См. [политики веб-служб](https://www.microsoft.com/licensing/terms/productoffering/MicrosoftDynamics365Services/EAEAS).)

**Номера SKU**

Полный список номеров SKU см. на вкладке "Номера SKU рекламной акции" в рекламной акции Dynamics 365 Cloud Migration и в [файле Excel с номерами SKU, которые перестают использоваться](https://partner.microsoft.com/resources/detail/dynamics-365-cloud-promotion-retired-skus-xls).

**Ресурсы**

- Цены и лицензирование Dynamics 365 для партнеров: [PartnerSource Business Center (PSBC)](https://businesscenter.mbs.microsoft.com/#contentdetail/Dyn365PricingandLicensing) (требуются действительные учетные данные)

- [Инструкции по переходу для Dynamics 365](https://mbs2.microsoft.com/fileexchange/?fileID=1324bd08-98ab-4de1-aa9d-4d9e8902c6a6)

- [Инструкции по переходу из локальной среды в облако для Dynamics 365](https://mbs2.microsoft.com/fileexchange/?fileID=53e8d8af-e8c5-4e7e-99b6-e81baa026275) (CSP)

- [Рекомендации по сценариям лицензирования Dynamics 365](https://mbs2.microsoft.com/fileexchange/?fileID=b82e7dad-46e5-475d-a23e-6bcee17cb5ea)

Требуется помощь? Обратитесь в [Сообщество партнеров Майкрософт](https://www.microsoftpartnercommunity.com/t5/Pricing-Licensing-Incentives/bd-p/PricingLicensingIncentives).
