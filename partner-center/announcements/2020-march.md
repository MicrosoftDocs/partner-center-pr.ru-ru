## <a name="microsoft-will-rename-office-365-small-and-medium-business-smb-offers-from-april-21-2020"></a><a id="7"/></a>21 апреля 2020 г. корпорация Майкрософт переименует предложения Office 365 для малых и средних предприятий (SMB)

### <a name="categories"></a>Категории

- Дата: 30.03.2020
- Введите команду: Предложения и рынки
- Состояние Ожидается в ближайшее время

### <a name="summary"></a>Сводка:

21 апреля 2020 г. корпорация Майкрософт переименует все предложения Office 365 для SMB. Это отличная возможность более согласованно информировать наших клиентов о выгодных предложениях и линейках продуктов.

### <a name="impacted-audience"></a>Затронутая аудитория

Все партнеры в рамках программы "Поставщик облачных решений" (CSP).

### <a name="details"></a>Подробности

21 апреля 2020 г. мы изменим имена предложений Office 365 для SMB:

- Предложение "Office 365 бизнес базовый" будет переименовано в **Microsoft 365 бизнес базовый**. Это наше предложение начального уровня для клиентов, которым нужны базовые облачные службы.
- Предложение "Office 365 бизнес премиум" будет переименовано в **Microsoft 365 бизнес стандартный**. Это наше основное предложение для SMB, включая облачные службы и классические приложения, подключенные к облаку.
- Предложение "Microsoft 365 бизнес" будет переименовано в **Microsoft 365 бизнес премиум**. Это основной номер SKU для предприятий SMB, использующих высокие технологии, которым нужна дополнительная защита, а также преимущества, касающиеся устройств и удостоверений.

Кроме того, новым названием для двух наших клиентских подписок Office станет **Приложения Microsoft 365**:

- Предложение "Office 365 бизнес" будет переименовано в "Приложения Microsoft 365 для бизнеса".
- Предложение "Office 365 ProPlus" будет переименовано в "Приложения Microsoft 365 для предприятий".

### <a name="impact-on-your-operations"></a>Влияние на вашу деятельность

Цены, функции и бизнес-модели останутся прежними. Также без изменений останутся имена SKU "Office 365 корпоративный" и "Microsoft 365 корпоративный". Все номера SKU и новые имена предложений будут автоматически отражены в названиях подписок в Центре партнеров.

### <a name="partner-opportunity"></a>Возможности для партнеров

Мы переименовываем предложения, чтобы клиенты могли составить более четкое представление о дополнительных преимуществах. Это откроет новые возможности для увеличения продаж в ваших учетных записях.

Кроме того, это изменение позволит устранить основные несоответствия имен в планах Office 365 и Microsoft 365 бизнес. Также переименование позволит четко отделить основные приложения Office, продаваемые в рамках планов "Приложения Microsoft 365 для бизнеса и предприятий", от решений, которые реализуются в рамках более широких планов.

### <a name="additional-resources"></a>Дополнительные ресурсы

Дополнительные сведения см. на нашей [целевой странице для партнеров M365](https://www.microsoft.com/microsoft-365/partners/smb-sku-rename).

### <a name="next-steps"></a>Дальнейшие действия

Ознакомьтесь с прейскурантом и передайте эти сведения всем соответствующим лицам в вашей организации.<br>

В особенности это касается отделов по операциям сбыта и маркетингу, так как, возможно, им потребуется обновить сопутствующие продукты, чтобы они отражали новые имена.

**Есть вопросы?**

Дополнительные сведения об этих предложениях можно получить в соответствующем сообществе партнеров в Yammer и у представителя службы поддержки учетных записей Майкрософт или в соответствующем сообществе партнеров в Yammer. Давайте поработаем вместе, чтобы добиться высоких результатов.

## <a name="new-partner-center-capabilities-to-help-direct-bill-partners-complete-their-transition-to-indirect-reseller"></a><a id="6"/></a>Новые возможности Центра партнеров, позволяющие партнерам с прямым выставлением счетов стать непрямыми торговыми посредниками.

### <a name="categories"></a>Категории

- Дата: 20.03.2020
- Введите команду: Характеристики
- Состояние Уже доступно
- [Подробнее](https://docs.microsoft.com/partner-center/)

### <a name="summary"></a>Сводка:

Появились новые возможности, позволяющие нашим клиентам передавать подписки CSP арендаторов с прямым выставлением счетов арендаторам непрямых поставщиков.

### <a name="impacted-audience"></a>Затронутая аудитория

Партнеры с прямым выставлением счетов, которые хотят стать непрямыми торговыми посредниками, а также непрямые поставщики.

### <a name="details"></a>Подробности

По состоянию на 31 августа 2018 г. корпорация Майкрософт обновила требования к регистрации для партнеров с прямым выставлением счетов в программе "Поставщик облачных решений" (CSP). Так как программа CSP продолжает развиваться, мы хотим убедиться в том, что у всех наших партнеров есть необходимые ресурсы для удовлетворения растущих потребностей клиентов.

Появились новые возможности, позволяющие клиентам передавать подписки CSP арендаторов с прямым выставлением счетов арендаторам непрямых поставщиков. Эта новая функциональная возможность упрощает переход со статуса партнера с прямым выставлением счетов на статус непрямого торгового посредника, помогая ускорить развитие бизнеса в рамках программы.

Дополнительные сведения см. на странице [предложений CSP](https://docs.microsoft.com/partner-center/csp-offers).

>[!NOTE] 
>При этом переходе будут поддерживаться только подписки на основе лицензий и подписки Azure. Вам нужно будет вручную перенести план Azure и покупки, сделанные на коммерческой платформе. Обратите внимание, что это подразумевает отмену и продление подписок, описанных в документах о [планах Azure](https://docs.microsoft.com/partner-center/azure-plan-lp) и [коммерческой платформе](https://docs.microsoft.com/partner-center/csp-commercial-marketplace-manage). Так как серверные подписки и резервирования Azure являются одноразовой покупкой, их не обязательно передавать арендатору непрямого поставщика. Чтобы получить дополнительные сведения, перейдите на страницу предложений CSP.

### <a name="additional-resources"></a>Дополнительные ресурсы:

- статья справки о [переходе со статуса партнера с прямым выставлением счетов на статус непрямого торгового посредника в рамках программы поставщиков облачных решений CSP](https://docs.microsoft.com/partner-center/transition-direct-to-indirect#move-your-direct-bill-customers-to-your-indirect-provider);
- [презентация о переходе со статуса партнера с прямым выставлением счетов на статус непрямого торгового посредника](https://assetsprod.microsoft.com/mpn/direct-bill-transition-to-indirect-reseller-new-capability).

### <a name="next-steps"></a>Дальнейшие действия

Изучите доступные ресурсы и поделитесь ими с соответствующими заинтересованными лицами в своей организации.

**Есть вопросы?**

Обратитесь в [службу поддержки](https://partner.microsoft.com/dashboard/support/csp/servicerequests/create?category=csp), если хотите задать вопрос, отправить запрос или получить дополнительные сведения.

## <a name="new-microsoft-365-entry-level-sku-for-firstline-workers---csp"></a><a id="5"/>Новый номер SKU Microsoft 365 начального уровня для работников первой линии в рамках программы "Поставщик облачных решений" (CSP)</a>

### <a name="categories"></a>Категории

- Дата: 10.03.2020
- Введите команду: Предложения и рынки
- Состояние Ожидается в ближайшее время

### <a name="summary"></a>Сводка:

1 апреля 2020 года корпорация Майкрософт представит новые предложения в линейке SKU Microsoft 365 F, которые привнесут дополнительные преимущества для работников первой линии и обеспечат нашим партнерам возможности для роста.

### <a name="impacted-audience"></a>Затронутая аудитория

Все партнеры, заключающие сделки в рамках Соглашений Enterprise (EA), Enterprise Agreement Subscription (EAS) и Microsoft Products and Services Agreement (MPSA).

Эти нововведения повлияют только на пользователей из коммерческого и государственного секторов.

### <a name="details"></a>Подробности

1 апреля 2020 года корпорация Майкрософт представит новый номер SKU начального уровня и изменит позиционирование линейки SKU Microsoft 365 корпоративный.

**Ребрендинг O/M365 F1.** Предложения Office и Microsoft 365 F1 будут переименованы в Office и Microsoft 365 F3. Все клиенты получат дополнительные преимущества.

**Новый номер SKU начального уровня.** Новый SKU начального уровня будет называться Microsoft 365 F1. Он предложит возможности, адаптированные для работников первой линии, а также расширенную безопасность и возможности идентификации для всех работников. Это предложение включает Teams, SharePoint Online, Yammer Enterprise, Stream, EMS E3, а также возможности доступа к мобильным и веб-приложениям Office в режиме чтения.

Эти изменения обеспечат дополнительные преимущества для работников первой линии и позволят партнерам существенно увеличить количество пользователей. **Дополнительные сведения о предлагаемых преимуществах и ваших дальнейших действиях см. [здесь](https://partner.microsoft.com/resources/collection/Microsoft-365-firstline-offer-updates)** .

|**График реализации**|**Изменение**|
|-------------------|:------|
|1 марта 2020 г.|Новый номер SKU начального уровня публикуется в предварительном прейскуранте. Временное название предложения — Microsoft 365 F0.|
|В течение марта|Майкрософт предоставит общедоступные сведения о предстоящих нововведениях. Тем временем вам нужно разработать маркетинговые материалы и подготовить внутренние отделы продаж.|

**1 апреля 2020 г. в планы M365 F1 и O/M365 F3 будут добавлены новые преимущества.**

### <a name="office"></a>Office.

- В SKU линейки F предложение Microsoft Teams включает полные права на аудио- и видеоматериалы, а также собрания (будет отражено в условиях использования продукта в марте).
- В Power Platform количество шагов увеличено до 2000, а также добавлена возможность создания (доступно только в M/O365 F3).
- Планировщик (Майкрософт) включен в линейку SKU O/M365 F по умолчанию.

### <a name="windows"></a>Windows.

- В стандартные условия использования продукта включено право на активацию с помощью системы управления ключами (KMS).
- В стандартные условия использования продукта включены права на предыдущие версии Windows.
- В стандартные условия использования продукта включены права на локальную виртуализацию.

### <a name="ems"></a>EMS.

- Предложение EMS E3 включено в SKU Microsoft 365 F1 и F3.

Ознакомьтесь с предварительным прейскурантом. Помните, что 1 апреля 2020 г. предложение Microsoft 365 F0 будет переименовано в Microsoft 365 F1, а Microsoft 365 F1 — в Microsoft 365 F3.

**Изучите [дополнительные сведения](https://partner.microsoft.com/resources/collection/Microsoft-365-firstline-offer-updates#/) о предлагаемых преимуществах** и создайте маркетинговые материалы, которые могут понадобиться для повышения продаж и создания новых бизнес-возможностей.

## <a name="updated-changes-to-education-customer-validation-in-csp"></a><a id="4"/></a>Обновление: изменения в проверке клиентов из образовательной сферы в рамках CSP

### <a name="categories"></a>Категории

- Дата: 10.03.2020
- Введите команду: Характеристики
- Состояние Уже доступно
- [Подробнее](https://partner.microsoft.com/resources/collection/csp-aer-partner-qualification-retirement#/)

### <a name="impacted-audience"></a>Затронутая аудитория

Партнеры и непрямые поставщики с прямым выставлением счетов, участвующие в программе CSP, которые работают с предложениями для образовательных и научных учреждений.

### <a name="summary"></a>Сводка:

Уведомление о новой функциональной возможности. Теперь партнеры могут исправлять сведения из учетной записи клиента и повторно отправлять их на проверку, если по клиенту пришел отказ. Подробные сведения о том, как это сделать, см. в обновленном пошаговом руководстве по [созданию клиента из сферы образования в Центре партнеров](https://assetsprod.microsoft.com/mpn/partner-center-education-customer-creation.pdf).

Кроме того, в рамках принятого обязательства перед клиентами и партнерами в отношении ведения бизнеса в соответствии с нормативными требованиями корпорация Майкрософт внедрила автоматическую проверку всех отправленных данных для новых учетных записей клиентов из сферы образования на предмет соответствия таких клиентов требованиям. С момента внедрения автоматической проверки мы наблюдаем возросшее число отказов, связанных с неправильными или недопустимыми данными о клиентах. Чтобы упростить процедуру и уменьшить задержки в ходе проверки, рекомендуем следовать приведенным ниже инструкциям.

- Чтобы сократить задержки при рассмотрении отказов, полученных по тем записям, которые отвечают требованиям, убедитесь, что вы указали максимально **точные** и **полные** данные о клиенте. Не вносите в систему тестовые или фиктивные данные.
- Изучите руководства по **созданию клиента из сферы образования в Центре партнеров** и **добавлению страницы нового клиента**, в которых подробно описываются требования к информации, которая необходима для создания нового клиента из сферы образования в Центре партнеров. Оба руководства можно найти в коллекции ресурсов ниже.

>[!NOTE] 
>Руководство по **созданию клиента из сферы образования в Центре партнеров** теперь содержит пошаговые инструкции по исправлению информации о клиенте и ее повторной отправке на рассмотрение на случай, если по клиенту пришел отказ.

**Дальнейшие действия и дополнительные ресурсы:**

Коллекция ресурсов касательно обновлений для программы CSP: [Изменения в программе для уполномоченных партнеров по образовательным учреждениям в рамках CSP](https://partner.microsoft.com/resources/collection/csp-aer-partner-qualification-retirement#/)

### <a name="change-log"></a>Журнал изменений

- Исходная дата публикации: 19 декабря 2019 г.
- Как обратиться в службу поддержки, если новый партнер, продающий предложения для образовательных учреждений, удален (больше не является обязательным): 13 января 2020 г.
- Добавлено обновление за февраль 2020 г.: 10 февраля 2020 г.

## <a name="microsoft-365-cloud-service-provider-updates--april-2020"></a><a id="3"/></a>Новости о Microsoft 365 в рамках программы "Поставщик облачных служб", апрель 2020 г.

### <a name="categories"></a>Категории

- Дата: 09.03.2020
- Введите команду: Предложения и рынки
- Состояние Ожидается в ближайшее время

### <a name="summary"></a>Сводка:

В апреле 2020 г. корпорация Майкрософт предложит целый ряд новых преимуществ для планов Microsoft 365.

### <a name="impacted-audience"></a>Затронутая аудитория

Все партнеры, заключающие сделки в рамках программы "Поставщик облачных решений" (CSP).

### <a name="details"></a>Подробности

1 апреля 2020 г. корпорация Майкрософт предложит целый ряд новых преимуществ для планов Microsoft 365. Благодаря этим шагам клиенты получат дополнительные возможности, а партнеры смогут расширить свою стратегию поставки продукции и услуг на рынок.

**Линейка SKU Microsoft 365 F: новые предложения и изменение позиционирования**

Корпорация Майкрософт представит новый номер SKU начального уровня для работников первой линии, что позволит партнерам существенно увеличить количество пользователей. Вследствие этого мы также изменяем позиционирование старого плана M365 F1. Теперь он будет называться M365 F3 и по умолчанию будет предоставлять дополнительные преимущества.

**Ознакомьтесь с дополнительными ресурсами по этой теме [здесь](https://partner.microsoft.com/resources/collection/Microsoft-365-firstline-offer-updates#/).**

### <a name="microsoft-365-for-unattended-scenarios"></a>Microsoft 365 для автоматизированных процессов
Лицензия Microsoft 365 E3 — Unattended позволяет использовать приложения Office с учетными записями служб или другими учетными записями, которые не принадлежат конкретным пользователям, как это обычно происходит во время выполнения процесса автоматизации. Майкрософт отменяет предварительное требование, согласно которому для использования предложения "Расширенная защита от угроз в Microsoft Defender" (MDATP) требовалась лицензия Windows E3. Теперь MDATP можно использовать на всех поддерживаемых клиентских устройствах.
