# <a name="june-2021-announcements"></a>Объявления за июнь 2021 г.

Эта страница содержит объявления, касающиеся Центра партнеров Майкрософт, за июнь 2021 г.

## <a name="phase-2-update-euefta-and-uk-change-of-partner-billing-currency-for-new-commerce-offers"></a><a name="9"></a>Новости этапа 2. По новым коммерческим предложениям партнеры из ЕС/ЕАСТ и Соединенного Королевства будут получать счета в своей валюте

### <a name="categories"></a>Категории

- Дата: 24.06.2021
- Возможности

### <a name="summary"></a>Сводка

В ближайшее время начнется этап 2 изменения валюты для выставления счетов партнеров по новым коммерческим предложениям для партнеров из Европейского союза (ЕС), Европейской ассоциации свободной торговли (ЕАСТ) и Соединенного Королевства.

### <a name="impacted-audience"></a>Затронутая аудитория

Партнеры, заключающие сделки в рамках программы "Поставщик облачных решений" (CSP) в регионах ЕС, ЕАСТ и в Соединенном Королевстве

### <a name="details"></a>Сведения

В ЕС/ЕАСТ и Соединенном Королевстве для всех новых коммерческих предложений в рамках программы CSP при выставлении счетов будет учитываться расположение партнера, а не расположение клиента. Это означает, что корпорация Майкрософт будет выставлять партнерам счета в местной валюте партнера, а не в местной валюте клиента. Мы объявили о [начале реализации этапа 1](./2021-january.md#15) 28 января 2021 г. Теперь мы готовы поделиться дополнительными сведениями об этапе 2 этого процесса перехода.

#### <a name="phase-2"></a>Этап 2

- Существующие клиенты, которые уже приобрели новое коммерческое предложение в рамках программы CSP до начала этапа 1.

- Существующие клиенты, которые приобрели новые коммерческие предложения впервые и арендатор для которых был создан не ранее 11 мая 2020 г.

**Обновление.** Корпорация Майкрософт отложила переход на этап 2 к середине августа, так как требуется дополнительное тестирование.

Начиная с середины августа 2021 г. корпорация Майкрософт переведет партнеров, существующие клиенты которых приобрели новое коммерческое предложение в рамках программы CSP до начала этапа 1, а также клиентов, для которых были созданы арендаторы не ранее 11 мая 2020 г., на выставление счетов за новые коммерческие предложения в местной валюте партнера.

После этапа 2 для партнеров в регионах ЕС, ЕАСТ и в Соединенном Королевстве счета для всех клиентов и за все покупки в рамках CSP будут выставляться в местной валюте партнера.

Этап 2 будет разворачиваться для партнеров постепенно. Это означает, что точная дата начала этапа 2 в организациях партнеров будет различаться. В ближайшие недели мы свяжемся с партнерами напрямую и сообщим им дополнительные сведения об их дате перехода.

Обратите внимание, что это изменение повлияет только на валюту выставления счетов для партнера. Цены на новые коммерческие предложения в рамках программы CSP не изменятся.Новые коммерческие предложения, на которые распространяется это изменение, включают:

- Подписки Azure, которые являются частью плана Azure
- Резервирования Azure
- Серверные подписки
- Программное обеспечение с бессрочной лицензией
- Коммерческая платформа Майкрософт

#### <a name="partneraction"></a>Действие партнера

Для подготовки к этому переходу мы описали ожидаемые действия партнеров в [руководстве](https://partner.microsoft.com/resources/detail/eu-efta-uk-change-of-partner-billing-currency-new-commerce-offers-pdf) (требуется вход в систему).

Никаких технических изменений в API Майкрософт или Центре партнеров не производится. Тем не менее, если вы используете API Майкрософт или являетесь сторонним поставщиком панели управления (CPV) для любой автоматизации выставления счетов, вам может потребоваться выполнить действие партнера, чтобы обеспечить возможность изменения валюты в вашей коммерческой системе.

#### <a name="partnerbenefits"></a>Преимущества для партнеров

Это обновление устранит сложности и накладные затраты, связанные с выставлением счетов в разных валютах в регионах ЕС, ЕАСТ и Соединенном Королевстве, для новой коммерческой модели.

- Партнеры будут получать консолидированный счет в одной валюте, а не несколько счетов в каждой местной валюте клиента.
- Выплаты по программам поощрений будут производиться в той же валюте, в которой партнеру выставляется счет.
- Партнеры заметят уменьшение сложности, связанной с выставлением счетов в нескольких валютах, и освобождение времени и ресурсов, которые были связаны с выверкой счетов.
- Для партнеров, которые еще не приняли новые коммерческие предложения, это изменение согласовано с предыдущей моделью выставления счетов, что позволяет партнерам более легко перейти на новые коммерческие модели в рамках программы CSP.

## <a name="phase-1-changes-to-azure-offer-availability-in-the-cloud-solution-provider-csp-related-to-the-new-commerce-experience-coming-soon"></a><a name="7"></a>Этап 1. Изменения в доступности предложений Azure в рамках программы "Поставщик облачных решений" (CSP), связанные со скорым выпуском новой коммерческой модели

### <a name="categories"></a>Категории

- Дата: 21.06.2021
- Возможности

### <a name="summary"></a>Сводка

Этап 1 изменений в доступности предложений Azure в рамках программы CSP, связанных с новой коммерческой моделью, начнется 21 июля 2021 г.

### <a name="impacted-audience"></a>Затронутая аудитория

Партнеры, работающие по программе CSP.

### <a name="details"></a>Подробнее

Как было объявлено в [январе](./2021-january.md#11), изменения предыдущего предложения Azure (плана Azure) в рамках программы CSP вступят в силу 21 июля 2021 г. с началом этапа 1.

Изменения в предыдущее предложение Azure будут вноситься в три этапа. На этапе 1 вся новая бизнес-деятельность в рамках программы Azure CSP будет переведена на новый план Azure. Приведенные ниже рекомендации относятся к следующему:

- Для всех существующих посреднических отношений между партнерами и клиентами в рамках программы CSP: если клиент уже приобрел предыдущее предложение Azure, то партнер может продолжать заключать сделки на предыдущее предложение Azure, а клиент может по-прежнему приобретать его у этого партнера.

- Предыдущее предложение Azure больше не будет доступно для новых партнеров, ведущих деятельность в рамках CSP.

- Предыдущее предложение Azure больше не будет доступно для новых клиентов, приобретающих решения у партнера в рамках CSP.

- Для всех новых посреднических отношений или новых приобретений лицензий клиентами: партнеры могут вести деятельность только в рамках нового предложения Azure (план Azure).

- Для всех существующих посреднических отношений между партнерами и клиентами: если клиент не приобретал Azure ранее, партнер может заключать сделки только в рамках нового предложения Azure, а клиент может приобретать у него только это предложение.

На этапе 2, который начнется в течение 2022-го календарного года, возможность поощрений и предоставления розничной скидки для партнеров будет удалена из предыдущего предложения Azure. Мы сообщим партнерам дату начала этапа 2 за шесть месяцев до ее наступления.

На этапе 3 все оставшиеся клиенты, использующие предыдущее предложение Azure, будут переведены на новое предложение Azure в рамках программы CSP (план Azure). Как и для предыдущих этапов, мы сообщим партнерам дату начала этапа 3 за шесть месяцев до ее наступления.

#### <a name="partner-earned-credit-pec"></a>Партнерские баллы (PEC)

Благодаря новой коммерческой модели партнеры получают вознаграждение за дифференцированные службы с добавленной ценностью на постоянной основе благодаря модели партнерских баллов. Партнерские баллы являются наградой для партнеров за помощь клиентам в достижении успеха благодаря созданию служб с добавленной ценностью, обеспечивающих устойчивую рентабельность.

## <a name="cloud-solution-provider-csp-promo-extensions-for-microsoft-365-audio-conferencing-and-microsoft-365-business-voice"></a><a name="5"></a>Продление акций в рамках CSP для Аудиоконференций Microsoft 365 и Голосовой связи Microsoft 365 бизнес 

### <a name="categories"></a>Категории

## <a name="perpetual-software-price-list-corrected-and-republished"></a><a name="4"></a>Прайс-лист на программное обеспечение с бессрочной лицензией был исправлен и опубликован заново

- Дата: 04.06.2021
- Предложения и рынки

### <a name="summary"></a>Сводка

Прайс-лист на программное обеспечение с бессрочной лицензией, опубликованный 1 июня 2021 г., был исправлен и опубликован заново 3 июня, чтобы устранить ряд ошибок. 

### <a name="impacted-audience"></a>Затронутая аудитория

Косвенные поставщики и партнеры с прямым выставлением счетов, которые предоставляют программное обеспечение с бессрочной лицензией в рамках программы "Поставщик облачных решений".  

### <a name="details"></a>Сведения

1. Неправильные цены в некоторых валютах.

   - Цены на некоторые продукты в российских рублях (RUB) и южнокорейских вонах (KRW) были указаны неправильно в период с 27 апреля 2021 г. по 3 июня 2021 г. Для этих продуктов были указаны некорректные цены, и они были исправлены в текущей версии прайс-листа на программное обеспечение с бессрочной лицензией.
 
     - Семейство продуктов Windows Server
     - Продукты SQL
     - Продукты Office
 
   - Проблемы с ценами были обнаружены достаточно рано, чтобы гарантировать правильное выставление счетов партнерам, которые оплатили по завышенной цене любые продукты с момента публикации неправильных цен. На счета затронутых партнеров будут направлены кредиты для корректировки ранее оплаченных счетов.

     - Если сумма оплаты в счете за май меньше суммы оплаты на 3 июня, партнерам будет выставлен счет по сумме оплаты за май.
     - Если сумма оплаты на 3 июня меньше суммы оплаты за май, партнерам будет выставлен счет по сумме оплаты за 3 июня.

2. SKUID для клиентской лицензии на устройство для продукта Project Server 2019 был исправлен с 0001 на 0003.  

3. Дополнительная вкладка под названием "Лист1(2)" на листе была удалена.


### <a name="next-steps"></a>Дальнейшие действия

- Партнеры, предлагающие программное обеспечение с бессрочной лицензией, должны скачать последнюю версию прайс-листа, чтобы получить правильные цены.

- Предложения и рынки

### <a name="impacted-audience"></a>Затронутая аудитория

Косвенные поставщики и партнеры с прямым выставлением счетов, которые предоставляют службы на основе лицензий в рамках программы "Поставщик облачных решений".

В предварительной версии прайс-листа за июль для служб на основе лицензий, опубликованной 1 июня 2021 г., была допущена опечатка, касающаяся предложений Office 365 E1 для некоммерческих организаций. Следующие предложения в прайс-листе были неправильно помечены как DEL (удаление). Прайс-лист для служб на основе лицензий был обновлен 4 июня 2021 г. в 11:00 PST с внесением соответствующих исправлений. Такие исправления применялись только к следующему:

- **Название предложения**: Office 365 E1 (цены для сотрудников некоммерческих организаций)
- **Идентификатор предложения**: 83dca3b9-3d6c-4547-8c5f-ba1f69b130f8 
- **Название предложения**: Office 365 E1 (цены для сотрудников некоммерческих организаций) — сотрудники, работающие на общественных началах
- **Идентификатор предложения**: f94de68e-f907-471f-b8d3-37ebacbe8ff6 

- **Валюта**: Все
- **До**: неправильно помечено как DEL в предварительной версии прайс-листа
- **После**: помечено как UNC в предварительной версии прайс-листа

Это исправление было связано только с прайс-листом и не влияло на выставление счетов.
