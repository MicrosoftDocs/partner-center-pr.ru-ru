## <a name="simplified-azure-apps-experience-in-azure-marketplace"></a><a name="8"></a>Упрощенный интерфейс приложений Azure в Azure Marketplace

### <a name="categories"></a>Категории

- Дата: 07.10.2020
- Функция
 
### <a name="summary"></a>Сводка 

Пользовательский интерфейс портала Azure был изменен для всех приложений Azure. Из этого новостного сообщения независимые поставщики программного обеспечения узнают о планах и требуемых действиях.

### <a name="impacted-audience"></a>Затронутая аудитория

Все независимые поставщики программного обеспечения, публикующие приложения Azure.

### <a name="details"></a>Подробнее

Мы упростили пользовательский интерфейс для приложений Azure на портале Azure.

Вы получили это сообщение, так как публиковали приложения Azure (шаблоны приложений и управляемые приложения) или зарегистрировались для их публикации в Azure Marketplace.

Мы внесли некоторые изменения в интерфейс [портала Azure](https://ms.portal.azure.com/), чтобы упростить для клиентов использование приложений Azure и управление ими. **Проверьте и при необходимости обновите** документацию, а также справочные материалы для клиентов для соответствия новому интерфейсу.
Новые возможности на портале Azure:

- гибкие возможности навигации, позволяющие просматривать, редактировать и сохранять информацию в любом разделе без внесения всех данных в предыдущих разделах;
- новое полностраничное представление без множества вложенных уровней;
- новые функции для организации ресурсов Azure и назначения им тегов (дополнительные сведения см. в статье об [упорядочении ресурсов с помощью тегов](https://docs.microsoft.com/azure/azure-resource-manager/management/tag-resources)).

Если у вас возникли вопросы об этих изменениях, обратитесь в нашу [службу технической поддержки](https://aka.ms/marketplacepublishersupport).

### <a name="next-steps"></a>Дальнейшие шаги

Проверьте и при необходимости обновите документацию, а также справочные материалы для клиентов для соответствия новому интерфейсу.

## <a name="microsoft-teams-rooms-premium-sku-launch"></a><a name="7"></a>Запуск номера SKU для Комнат Microsoft Teams категории "Премиум"

### <a name="categories"></a>Категории

- Дата: 05.10.2020
- Предложения и рынки
 
### <a name="summary"></a>Итоги 

Номер SKU для Комнат Microsoft Teams категории "Премиум" в рамках программы поставщиков облачных решений (CSP) станет доступен 1 ноября 2020 г. 

### <a name="impacted-audience"></a>Затронутая аудитория

Партнеры, работающие по программе CSP.

### <a name="details"></a>Подробнее

Новый номер SKU для Комнат Microsoft Teams категории "Премиум" уже доступен клиентам, приобретающим у вас облачные решения по программе CSP по цене 50 долл. США за устройство в месяц. Номер SKU для Комнат Microsoft Teams категории "Премиум" — это альтернатива номеру SKU для Комнат Microsoft Teams категории "Стандартный" (прежнее название — лицензия на Конференц-зал). Этот номер SKU включает все возможности предложения категории "Стандартный", например необходимые лицензии для Microsoft Teams, Skype для бизнеса Online и средства управления Intune. Это предложение также позволяет развернуть телефонную систему, необходимую для интеграции телефонной сети общего пользования (PSTN), и поддерживает возможности Аудиоконференций (при доступности).

С предложением категории "Премиум" клиенты получают доступ к новым службам Комнат Microsoft Teams, которые позволяют им передать управление конференц-залами и выполнение операций с ними соответствующим специалистам. Эта облачная служба для ИТ-управления и мониторинга поддерживает устройства Комнат Microsoft Teams и их периферийные устройства в актуальном и защищенном состоянии, а также в упреждающем режиме осуществляет мониторинг и управление, гарантируя удобство при проведении конференций.

#### <a name="microsoft-teams-rooms-managed-services"></a>Управляемые службы Комнат Microsoft Teams

- Упреждающее управление: непрерывное управление системами конференц-залов, в том числе установка исправлений, управление конфигурацией и многое другое.
- Мониторинг в реальном времени и анализ первопричин: мониторинг и обнаружение с управляемым реагированием на инциденты, обеспечиваемым корпорацией Майкрософт (с привлечением клиента при необходимости).
- Управляемые обновления: администрирование и доставка обновлений для приложений, базы данных Windows и встроенного ПО.
- Поддержка клиентов: круглосуточная обработка заявок на поддержку нашим специализированным операционным центром с сопровождаемым устранением инцидентов для заявок, которые нельзя обработать автоматически.
- Аналитика и рекомендации: агрегированная аналитика по клиентам и отчеты о состоянии, использовании конференц-залов, собраниях по сети и тенденциях инцидентов.

## <a name="microsoft-calling-plans-are-now-available-in-new-markets"></a><a name="6"></a>Планы звонков Майкрософт теперь доступны на новых рынках

### <a name="categories"></a>Категории

- Дата: 02.10.2020
- Предложения и рынки

### <a name="summary"></a>Итоги

Мы в Майкрософт рады объявить о том, что планы звонков Майкрософт уже доступны на следующих новых рынках: Австрия, Дания, Италия, Португалия, Швейцария и Швеция.
 
### <a name="impacted-audience"></a>Затронутая аудитория

Прямые и косвенные поставщики облачных решений (CSP), а также косвенные торговые посредники

### <a name="details"></a>Подробности

Мы в Майкрософт рады объявить о том, что с 1 октября 2020 г. планы звонков Майкрософт стали доступны на следующих новых рынках: Австрия, Дания, Италия, Португалия, Швейцария и Швеция. Поддержка новых географических областей стала доступна для коммерческого, некоммерческого и международного государственного секторов, а также для облака сообщества для государственных организаций в рамках Соглашения Enterprise (EA) или Соглашения Enterprise Subscription (EAS), Соглашения о регистрации для решений для образования (EES), Web Direct и CSP. 
 
Все функции планов звонков Майкрософт доступны для планов внутренних и международных звонков от Майкрософт, включая планы внутренних звонков на 120, 240 и 1200 минут на пользователя в месяц (p/u/m), а также международный план с 1200 минутами для внутренних звонков p/u/m и с 600 минутами для международных звонков p/u/m. 
 
Ранее планы звонков Майкрософт стали доступны на следующих рынках: Бельгия, Германия, Ирландия, Испании, Канада, Нидерланды, Пуэрто-Рико (доступны только в рамках EA и EES), Соединенное Королевство, США и Франция (в Австралии и Японии тарифный план доступен через сторонних поставщиков).

### <a name="impacted-audience"></a>Затронутая аудитория

Все партнеры, заключающие сделки в рамках программы "Поставщик облачных решений" (CSP).

### <a name="details"></a>Подробности

1 ноября 2020 г. станет доступен номер SKU расширенного взаимодействия в Microsoft Teams по специальной цене со скидкой 35 %. Этот номер SKU можно приобрести в с 1 ноября 2020 г. до 31 января 2021 г.

#### <a name="frequently-asked-questions"></a>Часто задаваемые вопросы

**Что собой представляет надстройка расширенного взаимодействия?**
Это новый дополнительный номер SKU для Microsoft Teams, позволяющий клиентам расширить их возможности взаимодействия в любом из пакетов Microsoft 365 или Office 365.

**Сколько стоит этот SKU без скидки?**
Оценочная розничная цена для коммерческого использования составляет 12 долл. США на пользователя в месяц.

**Какие клиенты могут приобрести надстройку расширенного взаимодействия?**
Коммерческие, образовательные и некоммерческие организации, а также международные организации коммерческого государственного сектора.

**Через какие каналы можно приобрести надстройку расширенного взаимодействия?**
По соглашению или подписке Enterprise (EA или EAS), в рамках программы Enrollment for Education Solutions (EES) или CSP, а также Web Direct.

**Какие клиенты могут приобрести номер SKU расширенного взаимодействия по специальной цене?**
Только коммерческие клиенты.

**Как приобрести номер SKU расширенного взаимодействия по специальной цене?**
Этот номер SKU по специальной цене можно приобрести через EA или EAS либо в рамках программы CSP. 

**Где можно продавать это предложение?**
Ее можно продавать на всех текущих рынках.

**Каковы предварительные требования для использования расширенного взаимодействия?**
Microsoft Teams и любой пакет Microsoft 365 или Office 365.

Клиенты, которые хотят и дальше использовать устройства с Windows 7 в 2021 году, могут приобрести расширенные обновления для системы безопасности Windows 7 на второй год (с завершением 12 января 2022 г.) у партнеров CSP уже сейчас. Это поможет клиентам принять решение относительно перехода на Windows 10. Дополнительные сведения см. в [коллекции ресурсов](https://partner.microsoft.com/resources/collection/windows-esu-csp#/).

## <a name="new-skus-bring-the-power-of-project-cortex-and-more-to-customers"></a><a name="2"></a>Новые номера SKU предоставляют клиентам преимущества Project Cortex и многие другие возможности

### <a name="categories"></a>Категории

- Дата: 01.10.2020
- Предложения и рынки

### <a name="summary"></a>Итоги

### <a name="details"></a>Подробнее

Рады объявить, что 1 октября 2020 г. станут доступны новые номера SKU и предложения, которые помогут клиентам воспользоваться возможностями [Project Cortex](https://resources.techcommunity.microsoft.com/project-cortex-microsoft-365/) и Microsoft Search для упрощения поиска информации и управления ею для пользователей.  

**SharePoint Syntex (ранее Интеллектуальные службы содержимого)**  
SharePoint Syntex использует расширенные возможности ИИ для автоматизации записи, добавления тегов и управления содержимым. Решение создано на основе SharePoint, что дает возможности обработки больших объемов содержимого, извлечения метаданных для автоматизации процессов и высокоточных механизмов обеспечения безопасности и соответствия. Решение упрощает проверку знаний и процесс обнаружения, предоставляя интеллектуальное содержимое для вашей сети знаний. 

Номера SKU для SharePoint Syntex станут доступны в прейскурантах с 1 октября (в предварительной версии прейскуранта от 1 сентября были указаны цены на Интеллектуальные службы содержимого). В лицензионных условиях Майкрософт такие номера SKU могут указываться со старым именем (Интеллектуальные службы содержимого) до 1 ноября 2020 г.

**Соединители Microsoft Graph для Microsoft Search**  

Соединители Microsoft Graph позволяют клиентам индексировать элементы во внешних репозиториях для включения в результаты Microsoft Search. С 1 октября 2020 г.:

- Microsoft 365 E5 и Office 365 E5 будут предоставлять возможность индексировать до 500 элементов с помощью соединителей Microsoft Graph для службы поиска Майкрософт (не входит в A5).  
- Любой пользователь с набором лицензий или отдельной лицензией, включающей в себя план SharePoint или OneDrive, сможет просматривать результаты поиска с помощью соединителей Microsoft Graph для службы поиска Майкрософт. 
- Будет доступен новый **номер SKU дополнительной емкости соединителя Graph**, который позволит индексировать до 1 млн дополнительных элементов.

Программы поощрений CSP на 2021 финансовый год начнут действовать с 1 октября 2020 г. Руководства и краткие презентации по поощрениями на 2021 финансовый год уже доступны в Центре партнеров.
 
### <a name="impacted-audience"></a>Затронутая аудитория

Партнеры с правом на прямое выставление счетов, косвенные поставщики и косвенные торговые посредники в рамках программы CSP.

Программа поощрений CSP на 2021 финансовый год начнет действовать с 1 октября 2020 г. и продлится до 30 сентября 2021 г. Обновленные ставки по программе на 2021 финансовый год будут действовать до 1 октября 2020 г. Партнеры увидят соответствующие изменения в своих ежемесячных выплатах по программе поощрений с 15 декабря 2020 г.
