# <a name="csp-one-time-purchase-reconciliation-file-fields"></a>Поля файла сверки одноразовой покупки CSP

## <a name="using-the-recon-file"></a>Использование файла разведывательную
В таблице ниже приводятся описания и примеры значений для полей в файле сверки для одноразовых покупок CSP.

Дополнительные сведения о файлах сверки см. [в разделе Использование файлов сверки](use-the-reconciliation-files.md).

| Столбец | Описание | Образец значения |
| ------ | ----------- | ------------ |
| PartnerId | Уникальный идентификатор в формате GUID для определенной сущности выставления счетов. Не требуется для сверки. Одинаково во всех строках. | *0e195b37-4574-4539-bc42-0e539b9684c0* |
| CustomerId | Уникальный идентификатор Майкрософт для клиента в формате GUID. | *196e2273-9651-43a3-ba7e-7cbcd918fc40* |
| CustomerName | Название организации клиента, указанное в Центре партнеров. Этот столбец важен для согласования счета со сведениями о системе. | *Джонни современный Cust DE2* |
| CustomerDomainName | Доменное имя клиента. | *testcustomerdomain.onmicrosoft.com* |
| CustomerCountry | Страна, в которой находится ваш клиент. См. полный [список стран](./regional-authorization-overview.md) для вашего региона.  | *RU* |
| InvoiceNumber | Номер счета, связанный с файлом сверки.  | *G002297372* |
| MpnId | Идентификатор MPN партнера CSP. Дополнительные сведения см. [в разделе как выполнить детализацию по партнерам](./use-the-reconciliation-files.md#itemize-reconciliation-files-by-partner). | *6034453* |
| реселлермпнид | MPN Идентификатор торгового посредника записи для подписки. | *6048879* |
| OrderId | Уникальный порядковый идентификатор платформы выставления счетов Майкрософт. При обращении в службу поддержки может оказаться полезным указать заказ. Не используется для сверки. | *0ET2qaZvJGfF9wgSKnWzR5JLmhp10lOc1* |
| OrderDate | Дата размещения заказа. | *10/3/2020* |
| ProductId | Уникальный идентификатор продукта. | *DZH318Z0BNZ5* |
| SkuId | Уникальный идентификатор SKU. | *006G* |
| AvailabilityId | Уникальный идентификатор доступности. | *DZH318Z08B80* |
| SkuName | Имя SKU. | *Таблицы — LRS* |
| ProductName | Название продукта. | *Таблицы* |
| ChargeType | [Тип оплаты](./recon-file-charge-types.md) или корректировки. | *Создать* |
| UnitPrice | Цена за лицензию, опубликованная в прейскуранте на момент приобретения. Убедитесь, что это соответствует информации, хранящейся в системе выставления счетов во время сверки. | *0,045* |
| количество; | Число лицензий. Убедитесь, что это соответствует информации, хранящейся в системе выставления счетов во время сверки. | *1* |
| Промежуточный итог | Сумма до налога. Подытог должен быть равен оплачиваемому количеству, умноженному на действительную цену единицы. | *0* |
| TaxTotal | Сумма налогов на оплату. В зависимости от правил налогов на рынке и определенных обстоятельств. | *0* |
| Итог | Общая сумма равна промежуточному итогу и сумме налогов. | *0* |
| Валюта | Счет формируется в контексте валюты клиента. Это означает, что партнеры, взаимодействующие с клиентами, которые работают с разными валютами, получат счет для каждого типа валюты клиента.  | *EUR* |
| прицеаджустментдескриптион | Причины для корректировок цены за единицу. Это основные причины, но не ограничиваются определением действительной цены единицы. | *["15,0% от партнерского кредита для управляемых служб"]* |
| PublisherName | Издатель продукта.  | *Майкрософт* |
| PublisherId | Уникальный идентификатор, используемый центром партнеров для идентификации издателя. | *СТРОЧ* |
| SubscriptionDescription | Имя предложения службы, приобретенной клиентом, как указано в прейскуранте. Этот столбец является идентичным полем для указывает. | *План Azure* |
| SubscriptionId | Уникальный идентификатор подписки платформы выставления счетов Майкрософт. Не используется для сверки. Обратите внимание, что этот идентификатор не совпадает с ИДЕНТИФИКАТОРом подписки в консоли администратора партнера. | *307628f1-d9d2-f09c-ea1f-4183f0cae308* |
| ChargeStartDate | Дата оплаты центра партнеров за плату за подписку. Если подписка куплена с годовым сроком выставления счетов и планом ежемесячного выставления счетов, то в первом файле сверки это день, когда подписка будет приобретена. Начиная со следующего файла сверки, он будет увеличен на 30 дней. | *9/1/2020* |
| ChargeEndDate | Конечный день оплаты для цикла выставления счетов подписки. Если подписка куплена с годовым сроком выставления счетов и планом ежемесячного выставления счетов, то в первом файле сверки это 30-дневный день после приобретения подписки. Начиная со следующего файла сверки, он будет увеличен на 30 дней. | *30.09.2020* |
| термандбиллингцикле | Обязательство продолжать подписку во время покупки. | *Хранимые данные (ГБ/месяц)* |
| еффективеунитприце | Распределенная Цена за единицу для расчета стоимости цикла выставления счетов. Скидки, корректировки в днях выставления счетов и другие факторы определяют фактическую цену за единицу. Дополнительные сведения см. в разделе [Вычисление эффективных единиц цен](./effective-unit-price-calculation.md).  | *0,03825* |
| Единицах UnitType | Тип единицы измерения, в которой начислена единица измерения. | *1 ГБ/мес.* |
| Альтернативного имени пользователя | Альтернативный идентификатор элемента строки заказа, на который указывает ссылка. | *6dc5c039750a* |
| биллаблекуантити | Общее количество, на которое взимается плата.  | *0,005001* |
| биллингфрекуенци | План выставления счетов, выбранный во время покупки. | *СТРОЧ*  |
| PricingCurrency | Валюта в прайс-листе. | *USD* |
| пктобцексчанжерате | Обменный курс, применяемый для валюты цены к валюте выставления счетов. | *0,846202666* |
| пктобцексчанжератедате | Дата, на которую определяется Валюта ценообразования в валюте выставления счетов. | *30.09.2020* |
| метердескриптион | Описание счетчика.  | *Таблицы — хранимые данные LRS (ГБ/месяц)* |
| ReservationOrderId | Идентификатор заказа на резервирование. | *E21A6344E398FFC1C4D7...* |
