# <a name="analyze-your-leads---see-how-well-you-attract-target-customers-and-potential-referrals"></a>Анализ потенциальных клиентов. Оценка уровня привлекательности компании для целевых клиентов и потенциальных рефералов
<!--

-->

**Соответствующие роли**

- Администратор авторов

На странице « **интересы** » в разделе « **аналитика** » ссылок можно увидеть, как выполняются ваши ссылки. Регулярно просматривайте эти метрики, чтобы определять тенденции или области, требующие внимания, и двигаться к бизнес-целям.

Чтобы просмотреть данные интересов Insights, в меню центра партнеров перейдите в раздел **ссылки > анализ > интересов**.

## <a name="apply-filters"></a>Применение фильтров

В верхней части страницы " **интересы** " можно выбрать период времени, для которого необходимо отобразить данные. По умолчанию выбрано значение **3 месяца**, но можно выбрать данные за 6 месяцев или 1 год. Вы также можете выбрать значение **Другое**, чтобы просмотреть данные по всем рефералам за определенный период времени.

Нажмите кнопку "Фильтры", чтобы открыть панель, которая позволяет отфильтровать все данные на этой странице по имени клиента, стране, типу сделки, направлению сделки, имени решения и состоянию. Ниже приведены подробные сведения о таких фильтрах.

- **Имя клиента.** По умолчанию используется фильтр **Все**, однако вы можете ограничить объем отображаемых данных одним или несколькими клиентами.
- **Страна.** По умолчанию используется фильтр **Все**, однако вы можете ограничить объем отображаемых данных одной или несколькими странами.
- **Справочная программа**: значение по умолчанию — **ALL**, но можно ограничить данные конкретной справочной программой. Этот фильтр отображается только в том случае, если ваша организация зарегистрирована в справочной программе.
- **Состояние**: По умолчанию используется фильтр **Все**, однако вы можете ограничить объем отображаемых данных рефералами, содержащими один или несколько типов состояния рефералов, таких как "принято", "отклонено", "истек срок действия", "утеряно" и "заключено".
- **Тип квалификации**: значение по умолчанию — **ALL**, но вы можете ограничить данные ссылками, которые являются либо интересами Marketplace, либо квалифицированными интересами.

Сведения на всех приведенных ниже диаграммах отражают диапазон дат и любые выбранные фильтры с учетом приведенных ниже исключений. Некоторые разделы также позволяют применять дополнительные фильтры, например фильтр по конкретному решению.

## <a name="referrals-summary"></a>Сводка по рефералам

На этой карточке представлен обзор того, как работают ваши интересы.

На диаграмме показано общее количество посещений страниц, число обращений к кнопке действия (контактного лица), число интересов, созданных после щелчка на мероприятии, и всего интересов, выигранных за выбранный период времени.

Метрики процентного изменения (показаны красным или зеленым цветом со стрелкой) указывают разницу между **последним полным месяцем в выбранном диапазоне дат** и **первым полным месяцем в данном диапазоне**. Предположим, что текущей датой является 15 июня и вы выбрали фильтр **3 месяца**, чтобы отобразить данные за последние три месяца. В данном случае эти метрики будут показывать разницу между маем (последний полный месяц в выбранном периоде) и мартом (первый полный месяц в выбранном периоде). Выбранный диапазон дат — последние **3 месяца**, поэтому будут сравниваться данные за май и за март.

## <a name="conversion-funnel"></a>Воронка конверсии

В этом разделе отображается визуальный индикатор перехода ваших сделок от одного состояния к другому в ходе их жизненного цикла. Вы можете просмотреть весь жизненный цикл сделок с учетом их общей суммы, а также суммы в долларах США по основному сведению для этого раздела. Первый раздел помечается типом сделки, чтобы дать вам визуальный индикатор объема или значения по типу. Кроме того, есть раздел, посвященный **прошлому**, который используется для указания сделок, для которых вы приняли меры принять или отклонить их или пометить как выигранные или потерянные за период времени, выбранный для отчета. Вы можете применить фильтры для просмотра динамики по сделкам на различных этапах их жизненного цикла.

## <a name="comparison-charts"></a>Сравнительные диаграммы

Раздел "Сравнение" поможет сравнить данные, связанные с интересами, на основе нескольких измерений как на томе, так и в значении "интересы" в сводной таблице USD.
Три показателя, по которым вы можете сравнить данные

- Тип квалификации
- Рынки
- Справочная программа

Если выбран тип квалификации, можно сравнить производительность ссылок в отношении интересов и квалифицированных интересов Marketplace. Как для рынков, так и для программы-ссылки можно выбрать три разных варианта для сравнения производительности. На первой диаграмме, которая представляет собой линейчатую диаграмму, будут приведены данные с динамикой в месячном выражении по основному сведению, то есть по общей сумме заключенных сделок. Справа от каждой линейчатой диаграммы также отображается круговая диаграмма с распределением этих же данных по процентной доле.

## <a name="no-data"></a>Нет данных

Существует несколько причин, по которым вы получаете пустую диаграмму, как показано ниже, при обращении к аналитике интересов, как описано ниже.

- Отсутствуют данные по этому клиенту. Попробуйте создать сделки, чтобы заполнить этот отчет.
- Возникла проблема с подключением к сети. Проверьте подключение к Интернету и повторите попытку.
- Отсутствуют записи, соответствующие примененному фильтру. Попробуйте сбросить фильтры.
- Между изменением состояния интереса и тем же обновлением в отчете аналитики есть задержка. Проверьте отчет через 24 часа.
