# <a name="create-suspend-or-cancel-customer-subscriptions"></a>Создание, приостановка и отмена подписок клиентов

**Относится к**

- Центр партнеров
- Центр партнеров для Microsoft Cloud for US Government
- Партнеры CSP

**Соответствующие роли**

- Агент по администрированию
- Администратор выставления счетов
- Глобальный администратор
- Агент службы технической поддержки
- Агент по продажам

После регистрации клиентов в Центре партнеров можно продать им подписки на продукты в каталоге. Сюда входят продукты, опубликованные корпорацией Майкрософт, а также продукты SaaS (программное обеспечение как услуга), опубликованные сторонними поставщиками программного обеспечения (ISV) для [коммерческого рынка](https://azuremarketplace.microsoft.com/marketplace).

Некоторые предложения ограничены одной подпиской на каждого клиента. Чтобы просмотреть список ограниченных предложений, посетите страницу "Цены и предложения" в Центре партнеров.

> [!IMPORTANT]
> Как партнер в программе CSP, вы можете приобрести подписки SaaS **на основе лицензий** только от издателей ISV в центре партнеров. Это означает, что вы можете приобрести любое предложение SaaS **на основе лицензий** . издатель ISV стал доступным вам, включая [исключительные предложения](csp-commercial-marketplace-discover.md#learn-about-marketplace-exclusive-offers) , к которым у вас есть доступ. Для приобретения или управления другими, коммерческими предложениями от независимых поставщиков программных продуктов (например, **на основе использования**, лимитных или на основе потребления, включающих приложения Azure, контейнеры или виртуальные машины), необходимо посетить [портал управления Azure](https://portal.azure.com/). Дополнительные сведения см. в статье [приобретение продуктов для коммерческого рынка](csp-commercial-marketplace-purchase.md).

## <a name="create-a-new-subscription"></a>Создание подписки

1. Войдите в [панель мониторинга Центра партнеров](https://partner.microsoft.com/dashboard).

2. В меню Центра партнеров выберите **Клиенты**, а затем выберите клиента из списка.

3. Выберите **Добавить подписку**. На вкладке " **веб-службы** " будут показаны все доступные предложения SaaS Marketplace.

4. Чтобы просмотреть только определенные типы подписок, сделайте выбор в доступных фильтрах:
   - **Издатель**: выберите **Майкрософт** , чтобы просмотреть только предложения от корпорации Майкрософт или **партнера** , чтобы просмотреть продукты коммерческого рынка, опубликованные независимыми поставщиками программного обеспечения.
   - **Тип выставления счетов**: выберите тип оплаты по подписке, который вы хотите использовать: **Лицензия** или **Использование**. В разделе [Вопросы и ответы по новым возможностям выставления счетов](faq-about-new-billing-features.md) предоставлены сведения, которые помогут выбрать удобную периодичность выставления счетов.
   - **Категория**: выберите **предприятие**, **малый бизнес**или **пробная версия**. Дополнительные сведения о пробных подписках см. в разделе [Предложение клиентам пробных версий продуктов Майкрософт](offer-your-customers-trials-of-microsoft-products.md).

5. Выберите подписки на продукт, которые вы хотите приобрести для клиента. Отображаемые продукты зависят от типа сегмента клиента (образование, правительственные учреждения и т. д.) и примененных фильтров. Некоторые предложения, отображаемые в Marketplace, могут не всегда быть доступны конкретному клиенту или определенному партнеру CSP. Это может быть вызвано:

   - У клиента уже есть подписка на этот продукт, и допускается только один

   - Возможно, подписка клиента приостановлена (в этом случае вы можете повторно активировать подписку, а не приобрести новую).

   - Для предложений SaaS ISV может быть несколько причин, по которым предложение недоступно для приобретения. независимого поставщика программного обеспечения может не поддерживать страну или регион выставления счетов. возможно, ISV не сделает предложение доступным через программу CSP. или независимый поставщик программного обеспечения может сделать предложение [эксклюзивным](csp-commercial-marketplace-discover.md#learn-about-marketplace-exclusive-offers) только определенным партнерам CSP. Предложение ISV также может быть недоступно для языка в центре партнеров (например, в контейнерах или в некоторых предложениях на основе использования).  

6. Для каждой подписки, которую необходимо добавить, введите число лицензий (если необходимо) и выберите **Добавить в корзину**.

7. По завершении добавления подписок выберите **Проверка** и проверьте свой заказ.

8. После рассмотрения заказов и готовности приобрести эти подписки выберите **купить**.

9. После приобретения подписки для клиента произойдет следующее:

    - Чтобы проверить или изменить подписку, выберите имя подписки на странице **подписки** этого клиента. Здесь можно выбрать дополнительные лицензии, если они доступны, изменить количество лицензий или приостановить подписку.

    **Для подписок ISV (на основе лицензий):**
    - Вы получите ссылку на сайт издателя ISV. Эта ссылка поможет вам завершить настройку развертывания или учетной записи для подписки клиента.
      
    >[!NOTE]
    > Ни вы, ни клиент не получит сообщение электронной почты с инструкциями по завершению настройки или подготовки учетной записи для этого типа подписки ISV.)

    - Если ваша подписка поставляется с 30-дневной бесплатной пробной версией, бесплатный пробный период будет применен автоматически. Как партнер в программе CSP вы не можете отказаться от использования бесплатного пробного периода на предложениях, приобретенных для клиентов. По окончании бесплатного пробного периода срок действия подписки начнется, а подписка будет преобразована в состояние оплачена. Подписка будет обновляться в соответствии с тем же расписанием.

## <a name="suspend-or-cancel-a-subscription"></a>Приостановка и отмена подписки

Партнеры могут приостановить или отменить подписку по просьбе клиента либо в случаях неплатежа или мошенничества.

### <a name="suspend-a-subscription"></a>Приостановка подписки

При изменении статуса подписки на **Приостановленная** пользователи не смогут входить в систему или получать доступ к службам.

1. Войдите на [панель мониторинга](https://partner.microsoft.com/dashboard) Центра партнеров.

2. В меню Центра партнеров выберите **Клиенты**, а затем выберите клиента из списка.

3. Выберите подписку, которую необходимо изменить.

4. В разделе **Состояние** выберите **Приостановлено**. Затем **отправьте** изменения.

5. Все данные будут удалены, если подписка не будет повторно активирована в течение 90 дней, либо 90 дней плюс количество дней от момента открытия учетной записи до первого периода выставления счетов (не более 120 дней).

При приостановке подписки дата, отображаемая под кнопкой **Приостановлено**, указывает, когда автоматически истечет срок действия подписки, если вы не активируете ее повторно. Дополнительные сведения см. в разделе [часто задаваемые вопросы о новых функциях выставления счетов](faq-about-new-billing-features.md).

### <a name="cancel-a-subscription"></a>Отмена подписки

Вы можете отменить подписки SaaS на основе лицензий сторонних издателей программного обеспечения в [коммерческом рынке](csp-commercial-marketplace-overview.md)центра партнеров. При условии отмены в течение периода отмены вы получите полный возмещение.

Для ISV предлагает ежемесячные счета:

- Если после размещения заказа вы отменили менее 24 часов, вы получите полный кредит для следующего счета.

- Если после размещения заказа вы отменили позже, чем через 24 часа, отмена будет запланирована на момент обновления.

Для предложений, которые выставляются ежегодно:

- Если после появления заказа вы отменили менее 14 дней, вы получите полный кредит для следующего счета.

- Если вы отменили позже через 14 дней после размещения заказа, то отмена будет запланирована при обновлении.

После выполнения этих периодов больше не будет отображаться возможность отмены подписки.

> [!NOTE]
> Службы независимых поставщиков программного обеспечения (например, использующие виртуальные машины или контейнеры) не могут возвращать данные на основе использования и измерения. В качестве метода отмены можно отменить подготовку служб на основе использования. Так как плата взимается за использование, эти службы не могут получить возмещение.

Чтобы отменить подписку SaaS на основе лицензий от издателя ISV, выполните следующие действия:

1. Войдите на [панель мониторинга](https://partner.microsoft.com/dashboard) Центра партнеров.

2. В меню Центра партнеров выберите **Клиенты**, а затем выберите клиента из списка.

3. Выберите подписку, которую нужно отменить.

4. В столбце **состояние** выберите **Отмена**. Затем **отправьте** изменения.

5. Если появится диалоговое окно, заполните все необходимые сведения и нажмите кнопку **Отправить**.

6. Чтобы подтвердить отмену, выберите **Да, отмена**.

> [!NOTE]
> Вы также можете отменить подписку Azure Marketplace с помощью интерфейсов API. Дополнительные сведения см. в разделе [Отмена подписки Azure Marketplace](https://docs.microsoft.com/partner-center/develop/cancel-an-azure-marketplace-subscription).

### <a name="choose-whether-to-automatically-renew-a-commercial-marketplace-subscription"></a>Выберите, следует ли автоматически продлить подписку на продукты коммерческого Marketplace

По умолчанию активные подписки настроены на автоматическое продление по истечении периода действия подписки. Для [подписок на коммерческие продукты Marketplace](csp-commercial-marketplace-overview.md)можно дополнительно отказаться от автоматического продления подписки.

Чтобы отключить автоматическое продление действующей подписки на коммерческий рынок, выполните следующие действия.

1. Войдите на [панель мониторинга](https://partner.microsoft.com/dashboard) Центра партнеров.

2. В меню Центра партнеров выберите **Клиенты**, а затем выберите клиента из списка.

3. Выберите **Подписки**. Здесь перечислены все подписки на основе лицензий, приобретенные для клиента.

4. В столбце **Подписка** выберите подписку, которую необходимо изменить.

5. На странице сведения о подписке выберите раздел **состояние** и снимите флажок **Auto-продление** .

6. Нажмите кнопку **Submit** (Отправить).
