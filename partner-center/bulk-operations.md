# <a name="bulk-operations-for-co-sell-opportunities-using-comma-separated-value-csv-files"></a>Групповые операции для совместной продажи с помощью файлов значений с разделителями-запятыми (CSV)

**Соответствующие роли**

- Администратор авторов
- Пользователь с ролью "Рекомендации"

Групповые операции в центре партнеров помогают компании экспортировать и импортировать данные о возможностях совместной продажи. Перейдите на страницу **возможности совместной** работы, чтобы найти ссылки **Импорт** и **Экспорт** в верхнем правом углу заголовка страницы. Эти функции могут использовать пользователи, имеющие разрешения **администратора** и ссылки для разрешений **пользователя** .

> [!IMPORTANT]
> Действия по созданию и обновлению, выполненные с помощью операции импорта, необратимы. Будьте внимательны при изменении или создании большого количества записей. Только подмножество полей можно изменить после создания сделки. **Действия не будут разрешены после того, как любая сделка достигнет состояния терминала, например "Отклонено" или "Просрочено", "Победа" или "потеря".**

## <a name="export-co-sell-opportunities"></a>Экспорт возможностей совместной продажи

Ниже приведены сведения о функции экспорта.

- Вы можете экспортировать не **более 5000 записей** , нажав кнопку **Экспорт** .
- Загружаемые сделки будут основываться на уровнях доступа. Администраторы ссылок и пользователи-участники могут получить разные результаты, исходя из их области действия и включаемые членами группы в сделки. Дополнительные сведения о [разрешениях для ссылок](permissions-overview.md#manage-referrals).
- Функция экспорта принимает, чтобы учитывать текущую вкладку на странице возможностей совместной продажи и примененные фильтры.
- Будет создан CSV-файл со всеми данными, основанными на примененных фильтрах.
- Скачивание записей может занять до одной минуты.
- Не нужно дожидаться завершения действия скачивания. Даже если перейти к другим страницам в центре партнеров, файл будет скачан сразу после завершения функции экспорта.
- Вы можете повторно использовать скачанный файл, чтобы изменить сведения о сделках и отправить для обновления любых записей.

## <a name="import-co-sell-opportunities"></a>Импорт возможностей совместной продажи

- Вы можете создать или обновить не **более 1000 записей** с помощью функции импорта.
- Шаблон можно создать с нуля, загрузив шаблон со страницы импорт в центре партнеров.
- Вы также можете использовать функцию экспорта, чтобы скачать существующие записи и обновить их.
- Если файл содержит более 1000 записей, он не может быть обработан.
- После обработки файла сводка с числом созданных, обновленных и не обработанных ссылок будет отображаться в карточке файла последнего процесса.
- Вы можете скачать сведения об обработанных записях, исправить ошибки и отправить один и тот же файл для создания или обновления записей, которые завершились сбоем во время предыдущего запуска. **Удалите все успешные записи из файла перед отправкой исправленных записей, которые завершились сбоем в предыдущем запуске.**
- Чтобы добавить дополнительные решения, добавьте дополнительные столбцы рядом с решением 1 и используйте имя столбца в качестве решения X, где X представляет номер решения в решении. Например, решение 2, решение 3.
- В свою сделку можно добавить до 50 решений.
- Чтобы добавить дополнительных членов команды, добавьте дополнительные столбцы рядом с элементом группы 1 и используйте имя столбца в качестве члена группы X, где X представляет номер участника команды. Например, член группы 2, член группы 3.
- В свою сделку можно добавить до 50 членов группы.

> [!NOTE]
> Не нужно дожидаться завершения обработки. Сведения о последнем обработанном файле будут доступны для скачивания после завершения обработки. **При отправке файлов с 1000 записями может потребоваться до 10 минут.**
